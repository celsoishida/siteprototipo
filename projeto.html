Projeto

Nós desenvolvemos protótipos para pesquisadores que não são da área de tecnologia. Ou seja, um pesquisador que não é da área de tencologia precisa validar sua pesquisa de ponta através de um sistema web ou aplicativo mobile, através de uma parceria o projeto Fábrica de REA desenvolve um protótipo funcional para esta validação.

  • Projeto de Repositório de Libras
  • Projeto Bibliografia - Contos e Livros
  • Projeto Chatbot de GI
  • Projeto Gramática para surdos
