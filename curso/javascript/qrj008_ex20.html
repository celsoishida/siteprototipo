Exercício 20: Criando um Menu Dropdown Interativo

  • Descrição: Crie a estrutura HTML para um menu dropdown (um <button> para acionar e uma <ul> escondida inicialmente com as opções). Usando JavaScript, adicione um "event listener" de "click" ao botão. Ao clicar no botão, alterne a visibilidade da lista de opções (mostrando se estiver escondida e escondendo se estiver visível).
  • Conceitos Teóricos:
    • Como controlar a visibilidade de elementos usando a propriedade style.display.
    • A lógica de "toggle" (alternar entre dois estados).
    • A importância de estruturar o HTML de forma semântica para facilitar a manipulação com JavaScript.
