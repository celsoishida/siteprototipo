Exercício 16: Criando Elementos Dinamicamente com Eventos

  • Tarefa: Crie um botão com o ID "adicionarItem". Adicione um "event listener" de "click" a este botão. Cada vez que o botão for clicado, crie um novo elemento <li> com um texto diferente (por exemplo, "Item 1", "Item 2", "Item 3", etc.) e adicione-o a uma lista não ordenada (<ul>) existente no seu HTML (adicione um ID).
  • Conceitos Teóricos:
    • Combinando a criação dinâmica de elementos (document.createElement()) com a manipulação do DOM em resposta a eventos.
    • A necessidade de manter um controle ou contador para gerar conteúdo dinâmico diferente.
