Exercício 3: Operações Matemáticas Básicas

  • Descrição: Crie código javascript que declare duas variáveis numéricas. Calcule a soma, subtração, multiplicação e divisão dessas variáveis. Exiba os resultados no console, formatando as mensagens de forma clara (ex: "A soma é: ...").
  • Conceitos Teóricos:
    • Tipo de dados primitivos: number.
    • Operadores aritméticos: +, -, *, /.
    • Interpolação de strings (template literals) para formatar a saída. Vídeo: vjs003

Dica: 
console.log(`Números: ${numero1} e ${numero2}\n`);
// ou
console.log('Números: ' + numero1 + ' e ' + numero2 + '\n');
