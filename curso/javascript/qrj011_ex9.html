Exercício 9: Exibindo um Prompt para o Usuário

  • Tarefa: Crie um botão no seu HTML. Adicione um "event listener" de "click" a este botão. Quando o botão for clicado, use a função prompt() para exibir uma caixa de diálogo pedindo ao usuário para digitar seu nome. Se o usuário digitar um nome e clicar em "OK", exiba um alert() com a mensagem "Olá, [nome]!".
  • Conceitos Teóricos:
    • A função prompt(mensagem, valorPadrao) para exibir uma caixa de diálogo que solicita entrada do usuário.
    • Como obter o valor digitado pelo usuário retornado por prompt().
