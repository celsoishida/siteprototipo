Exercício 15: Transformando Elementos de um Array (map)

  • Descrição: Crie um array de números (ex: [1, 2, 3, 4, 5]). Use um loop para criar um novo array onde cada elemento é o dobro do valor correspondente no array original e imprima o resultado.
  • Conceitos Teóricos:
    • Transformação de elementos em arrays.
    • O padrão de mapeamento (similar ao método map()).
    • Criação de novos arrays mantendo a mesma estrutura, mas com valores transformados.
