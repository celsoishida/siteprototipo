Exercício 4: Criando Elementos Dinamicamente com Repetição

  • Tarefa:
    • Crie um botão "Gerar Quadrados" e uma <div> vazia no seu HTML.
    • Usando JavaScript, adicione um "event listener" ao botão.
    • Quando o botão for clicado, peça ao usuário para digitar um número (use prompt()).
    • Use um loop for para criar um número de elementos <div> (com uma classe CSS para definir um quadrado - largura, altura, cor de fundo) igual ao número digitado pelo usuário.
    • Adicione todos esses novos elementos <div> à <div> vazia do seu HTML.
  • Conceitos Teóricos:
    • Seleção de elementos do DOM.
    • Manipulação de eventos de clique.
    • Interação com o usuário (prompt).
    • Loops for para criar múltiplos elementos.
    • Criação de novos elementos HTML e definição de suas propriedades (incluindo className).
    • Adição de múltiplos elementos ao DOM.
