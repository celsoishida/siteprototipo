Exercício 1: Introdução ao localStorage - Salvando um Valor Simples

  • Descrição: Use o método localStorage.setItem() para salvar um par chave-valor simples, como chave: "nomeDoUsuario" e valor: "João".
  • Conceitos Teóricos:
    • Introdução ao localStorage como uma forma de armazenamento persistente no navegador.
    • O método localStorage.setItem('chave', 'valor') para salvar dados.
    • A natureza síncrona do localStorage.
