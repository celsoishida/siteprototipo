Exercício 3: Alterando a Cor de Fundo ao Clicar em um Elemento

  • Tarefa: Crie um elemento <div> no seu HTML com uma cor de fundo inicial (definida no CSS ou inline). Usando JavaScript, adicione um "event listener" para o evento de "click" neste <div>. Ao clicar, altere a cor de fundo do <div> para outra cor (ex: amarelo). Ao clicar novamente, volte para a cor original.
  • Conceitos Teóricos:
    • O evento de "click" em outros elementos além de botões.
    • A propriedade style.backgroundColor para manipular a cor de fundo de um elemento.
    • A lógica de "toggle" (alternar entre dois estados).
