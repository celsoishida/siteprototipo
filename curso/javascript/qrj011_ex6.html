Exercício 6: Usando setInterval para Executar Código Repetidamente

  • Tarefa: Crie um elemento no seu HTML com o texto inicial "0". Use a função setInterval() para atualizar o texto do a cada segundo, incrementando um contador. Adicione um botão "Parar" que, ao ser clicado, use clearInterval() para interromper a atualização do contador.
  • Conceitos Teóricos:
    • A função setInterval(funcao, intervaloEmMilissegundos) para executar uma função repetidamente em intervalos regulares.
    • A função clearInterval(idDoIntervalo) para cancelar um intervalo definido por setInterval().
    • A importância de limpar intervalos para evitar vazamentos de memória.
