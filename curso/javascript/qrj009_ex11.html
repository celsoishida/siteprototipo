Exercício 11: Criando um Menu Dropdown Interativo

  • Descrição: Crie um menu dropdown interativo no seu HTML. O menu deve ter um botão principal com o texto "Menu" e, quando clicado, deve mostrar uma lista não ordenada (tag ul) de itens (pelo menos 3 itens). Use eventos para:
    • Mostrar/ocultar o menu quando o botão principal for clicado.
    • Fechar o menu automaticamente quando o usuário clicar fora dele.
    • Adicionar uma animação suave na transição de mostrar/ocultar (pode usar CSS transitions).
  • Conceitos Teóricos:
    • Manipulação do DOM para criar estruturas de menu.
    • Eventos de clique e eventos de fora do elemento (click fora).
    • Manipulação de classes CSS para controlar a visibilidade e animações.
    • Event delegation para lidar com múltiplos elementos.
    • Uso de CSS transitions para criar animações suaves.
