Exercício 10: Delegação de Eventos para Adicionar Comportamento a Múltiplos Elementos Dinâmicos

  • Descrição: Crie uma lista não ordenada (tag ul) vazia e um botão "Adicionar Item" no seu HTML. Usando JavaScript, adicione um "event listener" de "click" ao botão. Quando o botão for clicado, crie um novo item li com um texto (ex: "Item X") e um botão "Remover" dentro dele, e adicione este tag li à lista. Em vez de adicionar um "event listener" individual a cada botão "Remover" criado dinamicamente, adicione um único "event listener" de "click" à própria lista (tag ul). Dentro deste listener, verifique se o elemento alvo do clique (event.target) é um botão "Remover". Se for, remova o tag li pai desse botão.
  • Conceitos Teóricos:
    • O conceito de delegação de eventos (adicionar um único listener a um elemento pai para gerenciar eventos em seus filhos).
    • A propriedade event.target para identificar o elemento que disparou o evento.
    • Vantagens da delegação de eventos (performance, gerenciamento de elementos dinâmicos).
    • Navegação no DOM para acessar o elemento pai do alvo do evento.
