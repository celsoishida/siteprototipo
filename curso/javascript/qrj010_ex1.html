Exercício 1: Criando a Primeira Função

  • Descrição: Defina uma função chamada saudar que imprima a mensagem "Olá!" no console. Chame essa função para verificar se ela funciona.
  • Conceitos Teóricos:
    • O conceito de funções (blocos de código reutilizáveis).
    • A sintaxe básica para definir uma função usando a palavra-chave function, um nome e um bloco de código entre chaves {}.
    • Como chamar (invocar) uma função usando seu nome seguido de parênteses ().
