Exercício 10: Eventos de Input

  • Descrição: Crie um arquivo HTML com um campo de entrada de texto (<input type='text'>) e um parágrafo vazio. Crie código javascript JavaScript que, sempre que o usuário digitar algo no campo de entrada, atualize o texto do parágrafo com o que está sendo digitado.
  • Conceitos Teóricos:
    • Trabalhando com elementos de formulário (<input>).
    • O evento input que é disparado sempre que o valor de um elemento de entrada muda.
    • Acessando o valor de um campo de entrada usando a propriedade value.
