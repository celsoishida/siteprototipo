Exercício 8: Contador de Visitas usando localStorage

  • Tarefa: Crie um contador de visitas que:
    • Salve o número de visitas no localStorage usando localStorage.setItem().
    • Aumente o contador a cada visita e atualize o valor no localStorage.
    • Mostre o número atual de visitas na página.
  • Conceitos Teóricos:
    • Como usar localStorage para persistir dados entre sessões.
    • Como incrementar valores armazenados no localStorage.
    • Como exibir dados armazenados no localStorage na interface do usuário.
