Exercício 17: Removendo Elementos Dinamicamente com Eventos

  • Descrição: Modifique o exercício anterior para que cada <li> criado dinamicamente também contenha um botão "Remover" ao seu lado. Ao clicar neste botão "Remover", o item da lista correspondente deve ser removido do DOM.
  • Conceitos Teóricos:
    • A criação de múltiplos elementos dentro de um manipulador de eventos.
    • A necessidade de acessar o elemento pai (parentNode) para remover um elemento filho (removeChild()) dentro de um contexto dinâmico.
    • O conceito de delegação de eventos (embora não estritamente necessário para esta solução simples, é um bom ponto para introduzir posteriormente).
