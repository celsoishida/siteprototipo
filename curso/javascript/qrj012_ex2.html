Exercício 2: Recuperando um Valor de localStorage

  • Descrição: Use o método localStorage.getItem() para recuperar o valor associado à chave "nomeDoUsuario" salva no exercício anterior e imprima esse valor no console.
  • Conceitos Teóricos:
    • O método localStorage.getItem('chave') para recuperar dados.
    • Como o valor retornado é sempre uma string (ou null se a chave não existir).
