Exercício 9: Lembre-se da Última Escolha do Usuário com localStorage

  • Descrição: Crie um menu de temas (claro/escuro) que:
    • Salve a escolha do usuário no localStorage usando localStorage.setItem().
    • Recupere a escolha anterior quando a página for carregada usando localStorage.getItem().
    • Aplique automaticamente o tema escolhido anteriormente.
  • Conceitos Teóricos:
    • Como usar localStorage para persistir preferências do usuário.
    • Como recuperar e aplicar preferências ao carregar a página.
    • Como combinar localStorage com manipulação do DOM para aplicar estilos.
