Exercício 10: Pulando uma Iteração (continue)

  • Tarefa: Use um loop for para imprimir os números de 1 a 10. Se o número for 5, use continue para pular essa iteração e continuar com as outras.
  • Conceitos Teóricos:
    • A palavra-chave continue para pular a iteração atual e ir para a próxima.
    • Diferenças entre break e continue.
    • Casos de uso para continue em loops.
