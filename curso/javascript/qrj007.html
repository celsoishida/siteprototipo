G5 - Objetos em JavaScript

Olá! Aprender a trabalhar com objetos é fundamental em JavaScript, pois eles permitem organizar dados de maneira clara e estruturada. Os exercícios a seguir cobrem os conceitos fundamentais de objetos em JavaScript de forma progressiva, desde a criação básica até estruturas mais complexas como arrays de objetos e objetos aninhados.

  • Exercício 1: Criando o Primeiro Objeto Literal
  • Exercício 2: Acessando Propriedades com Notação de Ponto
  • Exercício 3: Acessando Propriedades com Notação de Colchetes
  • Exercício 4: Adicionando Novas Propriedades a um Objeto
  • Exercício 5: Modificando Valores de Propriedades Existentes
  • Exercício 6: Adicionando Métodos a um Objeto
  • Exercício 7: Usando this em Métodos
  • Exercício 8: Iterando sobre as Propriedades de um Objeto com for...in
  • Exercício 9: Arrays de Objetos
  • Exercício 10: Objetos Dentro de Objetos (Aninhamento)


Obs: Após o término dos exercícios, volte para o capítulo de 'DOM (Document Object Model)'.
