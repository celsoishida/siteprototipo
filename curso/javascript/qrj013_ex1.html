Exercício 1: Criando uma Lista de Tarefas Simples

  • Descrição:
    • Crie um input de texto e um botão "Adicionar" no seu HTML.
    • Usando JavaScript, adicione um "event listener" ao botão.
    • Quando o botão for clicado, capture o texto do input.
    • Crie um novo elemento <li> com o texto da tarefa.
    • Adicione este novo <li> a uma lista não ordenada (
        ) existente no seu HTML.
      • Limpe o valor do input após adicionar a tarefa.
    • Conceitos Teóricos:
      • Seleção de elementos do DOM (document.querySelector).
      • Manipulação de eventos (addEventListener).
      • Criação de novos elementos HTML (document.createElement).
      • Modificação do conteúdo de texto de um elemento (textContent).
      • Adição de elementos ao DOM como filhos de outro elemento (parentNode.appendChild).
      • Manipulação do valor de um input (element.value).
