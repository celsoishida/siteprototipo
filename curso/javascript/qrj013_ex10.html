Exercício 10: Criando um Sistema de Abas (Tabs) Simples

  • Descrição:
    • No seu HTML, crie uma série de botões (as abas) e várias <div>s com conteúdo correspondente a cada aba. Inicialmente, apenas o conteúdo da primeira aba deve estar visível (use CSS para esconder as outras). Associe cada botão a sua <div> de conteúdo (você pode usar atributos data-*).
    • Usando JavaScript, selecione todos os botões das abas e todos os conteúdos das abas.
    • Adicione um "event listener" de "click" a cada botão de aba.
    • Quando uma aba for clicada:
      • Esconda todos os conteúdos das abas.
      • Remova qualquer classe de "ativo" dos botões das abas.
      • Mostre o conteúdo da aba correspondente ao botão clicado (baseando-se no atributo data-*).
      • Adicione uma classe de "ativo" ao botão clicado para indicar que ele está selecionado (defina estilos para a classe "ativo" no CSS).
  • Conceitos Teóricos:
    • Seleção de múltiplos elementos.
    • Manipulação de eventos de clique.
    • Uso de atributos data-* para associar elementos.
    • Manipulação de estilos inline (style.display).
    • Adição e remoção de classes CSS (classList).
