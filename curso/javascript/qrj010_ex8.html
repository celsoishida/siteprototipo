Exercício 8: Funções com Valores Padrão para Parâmetros

  • Tarefa: Defina uma função chamada saudarComIdioma que aceite um parâmetro nome e um parâmetro opcional idioma com o valor padrão "Português". Dentro da função, imprima uma saudação no idioma especificado (ex: "Olá, [nome]!" para Português, "Hello, [nome]!" para Inglês). Chame a função passando apenas o nome e depois passando o nome e um idioma diferente.
  • Conceitos Teóricos:
    • Definindo valores padrão para parâmetros de funções usando a sintaxe parametro = valorPadrao.
    • Como os valores padrão são usados quando um argumento não é fornecido na chamada da função.
