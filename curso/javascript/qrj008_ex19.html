Exercício 19: Validando um Formulário Simples

  • Tarefa: Crie um formulário HTML com um campo de e-mail (<input type="email">) com o ID "email" e um botão de envio (<button type="submit">). Adicione um "event listener" para o evento de "submit" no formulário. Dentro do manipulador de eventos, impeça o envio padrão do formulário (event.preventDefault()) e verifique se o campo de e-mail está preenchido e se contém o caractere "@". Se a validação falhar, exiba uma mensagem de erro ao lado do campo de e-mail.
  • Conceitos Teóricos:
    • O evento de "submit" em formulários.
    • O método event.preventDefault() para impedir a ação padrão de um evento.
    • Como realizar validações básicas de entrada do usuário.
    • Como exibir mensagens de erro dinamicamente no DOM (criando ou modificando elementos).
