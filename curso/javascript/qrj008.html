G6 - DOM (Document Object Model) em JavaScript

Olá! O DOM (Document Object Model) é uma interface de programação que representa documentos HTML e XML como uma árvore de objetos. Com JavaScript, você pode manipular o DOM para criar páginas web dinâmicas e interativas. Os exercícios a seguir cobrem os conceitos fundamentais de manipulação do DOM, desde a seleção de elementos até a criação de interações complexas.
