Exercício 2: Manipulando Texto

  • Descrição: Crie código javascript que declare uma variável com o seu nome e outra com uma saudação (ex: "Bem-vindo(a)"). Concatene as duas strings e exiba o resultado em um alert().
  • Conceitos Teóricos:
    • Declaração de variáveis com let ou const.
    • Tipos de dados primitivos: string.
    • Operador de concatenação (+).
    • A função global alert() para exibir mensagens na tela.
