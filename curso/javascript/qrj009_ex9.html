Exercício 9: Arrastando e Soltando um Elemento (Básico)

  • Descrição: Crie um elemento div no seu HTML (com alguma cor de fundo para torná-lo visível).     • O objeto de evento (event) e as propriedades event.clientX e event.clientY para obter as coordenadas do mouse.
    • Manipulação das propriedades de posição (style.left, style.top) com posicionamento absolute ou fixed no CSS.
    • Uso de variáveis de controle (flags) para gerenciar o estado de arrasto.
