Exercício 3: Redirecionando para Outra Página

  • Descrição: Crie um botão no seu HTML. Adicione um "event listener" de "click" a este botão. Quando o botão for clicado, use o objeto window.location para redirecionar o navegador para outra página da web (ex: https://www.google.com).
  • Conceitos Teóricos:
    • O objeto window.location para interagir com a URL atual.
    • A propriedade window.location.href para definir ou obter a URL da página atual.
    • Redirecionamento de página via JavaScript.
