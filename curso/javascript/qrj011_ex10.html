Exercício 10: Exibindo uma Caixa de Confirmação

  • Descrição: Crie um botão no seu HTML. Adicione um "event listener" de "click" a este botão. Quando o botão for clicado, use a função confirm() para exibir uma caixa de diálogo com uma pergunta (ex: "Você tem certeza?"). Se o usuário clicar em "OK", imprima "Usuário confirmou!" no console. Se clicar em "Cancelar", imprima "Usuário cancelou." no console.
  • Conceitos Teóricos:
    • A função confirm(mensagem) para exibir uma caixa de diálogo de confirmação com botões "OK" e "Cancelar".
    • Como obter o valor booleano retornado por confirm() (true para "OK", false para "Cancelar").
