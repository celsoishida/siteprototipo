Exercício 1: Olá, Mundo!

  • Descrição: Crie um código em JavaScript que exiba a mensagem "Olá, Mundo!" no console do navegador.
  • Conceitos Teóricos:
    • Introdução ao JavaScript e sua função no desenvolvimento web.
    • A tag <script> para incluir código JavaScript em um documento HTML.
    • O objeto console e o método log() para exibir informações no console.
