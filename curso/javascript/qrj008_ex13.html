Exercício 13: Adicionando e Removendo Event Listeners

  • Tarefa: Crie um botão (<button>) no seu HTML com o ID "meuBotao". Usando JavaScript, adicione um "event listener" para o evento de "click". Quando o botão for clicado, exiba um alert() com a mensagem "Botão clicado!". Em seguida, remova esse "event listener".
  • Conceitos Teóricos:
    • O método element.addEventListener('tipoDoEvento', funcaoCallback) para adicionar um ouvinte de eventos a um elemento.
    • Funções de callback (a função que é executada quando o evento ocorre).
    • O método element.removeEventListener('tipoDoEvento', funcaoCallback) para remover um ouvinte de eventos. Importante: Para remover, a função de callback passada deve ser a mesma instância da função adicionada.
