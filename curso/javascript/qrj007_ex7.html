Exercício 7: Usando this em Métodos

  • Tarefa: Modifique o objeto aluno do exercício anterior. No método dizerOla, use a palavra-chave this para acessar a propriedade nome do objeto ao invés de usar aluno.nome diretamente. Adicione também um método chamado mostrarNota que imprima no console "Minha nota é [nota do aluno].". Use this para acessar a propriedade nota. Chame ambos os métodos.
  • Conceitos Teóricos:
    • A palavra-chave this dentro de um método de objeto, que se refere ao próprio objeto que chamou o método.
    • A importância de this para acessar as propriedades do objeto dentro de seus métodos.
    • Como this torna os métodos mais reutilizáveis e independentes do nome do objeto.
