G7 - Eventos em JavaScript

Olá! A interação com o usuário é um aspecto crucial do desenvolvimento web, e os eventos em JavaScript são a chave para isso. Aqui estão 10 exercícios graduais para o aprendizado de eventos:

  • Exercício 1: Exibindo um Alerta ao Clicar em um Botão
  • Exercício 2: Mudando o Texto de um Elemento ao Passar o Mouse
  • Exercício 3: Alterando a Cor de Fundo ao Clicar em um Elemento
  • Exercício 4: Capturando o Valor de um Input ao Digitar
  • Exercício 5: Focando e Desfocando um Input
  • Exercício 6: Enviando um Alerta ao Submeter um Formulário
  • Exercício 7: Alternando a Visibilidade de um Elemento com um Botão
  • Exercício 8: Adicionando um Item a uma Lista com Enter
  • Exercício 9: Arrastando e Soltando um Elemento (Básico)
  • Exercício 10: Delegação de Eventos para Adicionar Comportamento a Múltiplos Elementos Dinâmicos
  • Exercício 11: Criando um Menu Dropdown Interativo


Obs: Após o término dos exercícios, volte para o capítulo de 'Função'.
