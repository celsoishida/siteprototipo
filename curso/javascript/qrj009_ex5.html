Exercício 5: Focando e Desfocando um Input

  • Descrição: Crie um campo de texto ( <input type="text" /> )no seu HTML. Usando JavaScript, adicione "event listeners" para os eventos focus (quando o input recebe o foco) e blur (quando o input perde o foco). Ao receber o foco, adicione uma borda destacada ao input (manipule a propriedade style.border). Ao perder o foco, remova a borda ou volte ao estilo original.
  • Conceitos Teóricos:
    • Os eventos de formulário focus e blur.
    • Manipulação de estilos CSS através da propriedade style.
