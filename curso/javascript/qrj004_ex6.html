Exercício 6: Estruturas Condicionais

  • Descrição: Declare uma variável com uma idade. Use estruturas condicionais (if, else if, else) para classificar a idade em categorias: "Criança" (0-12 anos), "Adolescente" (13-17 anos), "Adulto" (18-59 anos) e "Idoso" (60+ anos). Exiba a categoria no console.
  • Conceitos Teóricos:
    • A estrutura condicional if.
    • A estrutura condicional if...else.
    • A estrutura condicional if...else if...else para verificar múltiplas condições em sequência.
    • A ordem das condições em uma estrutura if...else if.
