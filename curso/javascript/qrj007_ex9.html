Exercício 9: Arrays de Objetos

  • Tarefa: Crie um array chamado pessoas contendo pelo menos três objetos literais. Cada objeto deve ter as propriedades nome e idade. Use um loop for (ou for...of) para iterar sobre o array e imprimir o nome e a idade de cada pessoa no console.
  • Conceitos Teóricos:
    • Arrays que contêm objetos como seus elementos.
    • Como acessar as propriedades de objetos dentro de um array usando a combinação de indexação de array e notação de ponto ou colchetes.
    • Uso comum de arrays de objetos para representar coleções de dados estruturados.
