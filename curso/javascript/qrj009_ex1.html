Exercício 1: Exibindo um Alerta ao Clicar em um Botão

  • Tarefa: Crie um botão (<button>) no seu HTML. Usando JavaScript, adicione um "event listener" para o evento de "click" neste bot o. Quando o bot o for clicado, exiba um alert() com a mensagem "Bot o clicado!".
  • Conceitos Teóricos:
    • O conceito de eventos em JavaScript (ações que ocorrem no navegador).
    • O método addEventListener('tipoDoEvento', funcaoCallback) para registrar um ouvinte de eventos em um elemento.
    • O evento de "click".
    • A função alert() para exibir uma caixa de diálogo.
