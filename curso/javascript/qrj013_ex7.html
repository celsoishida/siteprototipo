Exercício 7: Adicionando e Removendo Itens de uma Lista com Botões Individuais

  • Descrição:
    • Crie uma lista não ordenada (<ul>) com alguns itens (
    • ) no seu HTML. Para cada
    • , adicione um botão "Remover" ao lado.
    • Usando JavaScript, adicione "event listeners" a cada botão "Remover".
    • Quando um botão "Remover" for clicado, identifique o
    • pai desse botão e remova-o da lista (parentNode.removeChild).
  • Conceitos Teóricos:
    • Seleção de múltiplos elementos (botões).
    • Iteração sobre elementos para adicionar event listeners.
    • Navegação no DOM para acessar o elemento pai (button.parentNode).
    • Remoção de elementos do DOM (parentNode.removeChild).
