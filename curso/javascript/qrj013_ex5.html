Exercício 5: Alternando a Visibilidade de um Elemento com um Botão

  • Descrição:
    • Crie um elemento <div> com algum conteúdo e um botão "Mostrar/Esconder" no seu HTML. Inicialmente, o <div> pode estar visível.
    • Usando JavaScript, adicione um "event listener" ao botão.
    • Mantenha o controle do estado de visibilidade do <div> (visível ou escondido).
    • Quando o botão for clicado:
      • Se o <div> estiver visível, esconda-o alterando sua propriedade style.display para "none" e mude o texto do botão para "Mostrar".
      • Se o <div> estiver escondido, mostre-o alterando sua propriedade style.display para "" (ou "block", "inline-block", dependendo do elemento) e mude o texto do botão para "Esconder".
  • Conceitos Teóricos:
    • Seleção de elementos do DOM.
    • Manipulação de eventos de clique.
    • Manipulação de estilos inline (element.style.display).
    • Uso de variáveis para controlar o estado da aplicação.
    • Lógica condicional (if...else) para alterar o comportamento com base no estado.
    • Modificação do conteúdo de texto de um elemento (textContent).
