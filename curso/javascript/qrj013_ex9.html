Exercício 9: Filtrando uma Tabela Dinamicamente

  • Tarefa:
    • Crie uma tabela HTML com cabeçalhos e algumas linhas de dados. Adicione um input de texto acima da tabela com o rótulo "Filtrar".
    • Usando JavaScript, selecione o input de texto e as linhas da tabela (excluindo o cabeçalho).
    • Adicione um "event listener" de "input" ao campo de texto.
    • A cada vez que o valor do input mudar, percorra todas as linhas da tabela.
    • Para cada linha, verifique se o texto de alguma das células (você pode escolher uma coluna específica para filtrar ou todas) contém o texto digitado no input (ignorando maiúsculas e minúsculas).
    • Se a linha não contiver o texto de filtro, esconda-a (style.display = "none"). Caso contrário, mostre-a (style.display = "").
  • Conceitos Teóricos:
    • Seleção de elementos de tabela (, , , ,
      ).
    • Manipulação de eventos de input.
    • Iteração sobre coleções de elementos (linhas e células da tabela).
    • Manipulação de estilos inline para controlar a visibilidade.
    • Métodos de string para verificar se um texto contém outro (includes(), toLowerCase()).
