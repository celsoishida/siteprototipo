Exercício 8: Manipulando o DOM (Parte 1)

  • Descrição: Crie um arquivo HTML com um parágrafo (<p>) com algum texto e um botão (<button>) com o texto "Clique Aqui". Crie código JavaScript que, ao clicar no botão, altere o texto do parágrafo para "Você clicou no botão!".
  • Conceitos Teóricos:
    • Introdução ao Document Object Model (DOM).
    • Selecionando elementos HTML usando document.getElementById() (adicione um id ao parágrafo e ao botão no HTML).
    • Adicionando um ouvinte de eventos (addEventListener) a um elemento.
    • Manipulando o conteúdo de um elemento HTML (textContent).
