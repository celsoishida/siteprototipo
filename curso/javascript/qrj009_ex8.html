Exercício 8: Adicionando um Item a uma Lista com Enter

  • Descrição: Crie um input de texto e uma lista não ordenada (tag ul) vazia no seu HTML. Usando JavaScript, adicione um "event listener" para o evento de "keypress" no input de texto. Quando a tecla "Enter" (código da tecla 13) for pressionada, capture o valor do input, crie um novo elemento
  • com esse valor e adicione-o à lista. Limpe o input após adicionar o item.
  • Conceitos Teóricos:
    • O evento de teclado keypress.
    • O objeto de evento (event) e a propriedade event.key ou event.keyCode para identificar a tecla pressionada.
    • Criação e adição dinâmica de elementos ao DOM.
