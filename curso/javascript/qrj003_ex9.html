Exercício 9: Manipulando o DOM (Parte 2)

  • Descrição: Crie um arquivo HTML com uma lista não ordenada (<ul>) vazia. Crie código JavaScript que adicione dinamicamente três itens de lista (<li>) a essa lista, cada um contendo o nome de uma cor diferente.
  • Conceitos Teóricos:
    • Criando novos elementos HTML com document.createElement().
    • Adicionando conteúdo a um elemento com textContent.
    • Adicionando um elemento filho a outro elemento com appendChild().
    • Selecionando elementos HTML usando document.querySelector() ou document.querySelectorAll().
