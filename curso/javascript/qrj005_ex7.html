Exercício 7: while com Condição Baseada em Input

  • Tarefa: Peça ao usuário para digitar um número. Use um loop while para continuar pedindo até que o usuário digite um número maior que 10.
  • Conceitos Teóricos:
    • Loops while com condições que dependem de interação do usuário.
    • Uso do prompt() para obter entrada do usuário.
    • Validação de entrada usando loops.
