G10 - Storage em JavaScript

Olá! O armazenamento no navegador permite que as aplicações web persistam dados localmente no navegador do usuário, melhorando a experiência e permitindo funcionalidades offline. Os exercícios seguintes proporcionam uma introdução prática e gradual ao armazenamento no navegador usando localStorage e sessionStorage, cobrindo os conceitos básicos de salvar, recuperar, remover e limpar dados, além de cenários de uso comuns.

  • Exercício 1: Introdução ao localStorage - Salvando um Valor Simples
  • Exercício 2: Recuperando um Valor de localStorage
  • Exercício 3: Removendo um Item de localStorage
  • Exercício 4: Limpando todo o localStorage
  • Exercício 5: Salvando e Recuperando um Objeto JSON
  • Exercício 6: Introdução ao sessionStorage - Salvando e Recuperando Dados de Sessão
  • Exercício 7: Removendo e Limpando o sessionStorage
  • Exercício 8: Contador de Visitas usando localStorage
  • Exercício 9: Lembre-se da Última Escolha do Usuário com localStorage
  • Exercício 10: Persistindo um Pequeno Formulário com localStorage


Obs: Após o término dos exercícios, volte para o capítulo de 'Requisições HTTP'.
