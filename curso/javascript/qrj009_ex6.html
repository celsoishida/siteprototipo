Exercício 6: Enviando um Alerta ao Submeter um Formulário

  • Descrição: Crie um formulário (<form>) no seu HTML com pelo menos um campo de input e um botão de submit (<button type="submit">). Usando JavaScript, adicione um "event listener" para o evento de "submit" no formulário. Dentro da função de callback, use event.preventDefault() para impedir o envio padrão do formulário e, em seguida, exiba um alert() com a mensagem "Formulário enviado (de mentirinha)!".
  • Conceitos Teóricos:
    • O evento de formulário submit.
    • O objeto de evento (event) e o método event.preventDefault() para impedir o comportamento padrão de um evento.
