Exercício 7: Removendo e Limpando o sessionStorage

  • Tarefa: Use sessionStorage.removeItem() para remover o item "ultimoProduto" e sessionStorage.clear() para limpar todo o sessionStorage. Verifique que os dados foram removidos usando sessionStorage.getItem().
  • Conceitos Teóricos:
    • Como os métodos de remoção e limpeza funcionam de forma similar ao localStorage, mas aplicados ao sessionStorage.
    • Como verificar se os dados foram removidos corretamente.
