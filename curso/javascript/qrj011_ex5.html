Exercício 5: Usando setTimeout para Executar Código Após um Tempo

  • Tarefa: Crie um botão no seu HTML. Adicione um "event listener" de "click" a este botão. Quando o botão for clicado, use a função setTimeout() para exibir um alert() com a mensagem "Olá após 3 segundos!" após um atraso de 3000 milissegundos.
  • Conceitos Teóricos:
    • A função setTimeout(funcao, atrasoEmMilissegundos) para executar uma função uma vez após um determinado atraso.
