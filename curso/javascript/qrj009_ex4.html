Exercício 4: Capturando o Valor de um Input ao Digitar

  • Tarefa: Crie um campo de texto (<input type="text">) no seu HTML. Usando JavaScript, adicione um "event listener" para o evento de "input". A cada vez que o valor do input mudar, imprima o valor atual no console usando console.log().
  • Conceitos Teóricos:
    • O evento de "input" (disparado sempre que o valor de um elemento de formulário muda).
    • A propriedade value de um elemento de input para acessar o seu valor.
