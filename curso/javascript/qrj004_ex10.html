Exercício 10: Operadores Lógicos Avançados

  • Descrição: Crie um programa que simule um sistema de login. Declare variáveis para armazenar um nome de usuário e uma senha. Use operadores lógicos para verificar se o nome de usuário é válido (não vazio) E a senha tem pelo menos 6 caracteres OU contém um caractere especial. Exiba mensagens apropriadas no console.
  • Conceitos Teóricos:
    • Combinação de operadores lógicos em expressões complexas.
    • Precedência de operadores lógicos.
    • Uso de parênteses para controlar a ordem de avaliação.
    • Aplicação prática de operadores lógicos em validações.
