Exercício 18: Filtrando uma Lista com Input

  • Descrição: Crie uma lista não ordenada (<ul>) com vários itens (<li>) no seu HTML. Adicione um campo de texto (<input type="text">) com o ID "filtro". Adicione um "event listener" de "input" (que dispara a cada mudança no valor do input) ao campo de texto. À medida que o usuário digita no campo de texto, filtre a lista, mostrando apenas os itens que contêm o texto digitado (ignorando maiúsculas e minúsculas).
  • Conceitos Teóricos:
    • O evento de "input".
    • Como iterar sobre uma coleção de elementos (o NodeList retornado por querySelectorAll).
    • A propriedade textContent para comparar o texto dos itens da lista com o valor do input.
    • Como controlar a exibição de elementos usando a propriedade style.display (definindo para "none" para ocultar e "" para exibir).
