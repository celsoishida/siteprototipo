Exercício 2: Selecionando Elementos por ID

  • Descrição: Crie um elemento HTML com um ID específico (por exemplo,
    Olá!
    ) e, usando JavaScript, selecione esse elemento pelo seu ID e altere seu conteúdo de texto.
  • Conceitos Teóricos:
    • O método document.getElementById() para selecionar elementos pelo atributo ID.
    • A propriedade textContent para acessar ou modificar o conteúdo de texto de um elemento.
    • A importância de IDs únicos no documento HTML.
