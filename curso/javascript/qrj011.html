  • Exercício 2: Obtendo as Dimensões da Janela
  • Exercício 3: Redirecionando para Outra Página
  • Exercício 4: Abrindo uma Nova Janela
  • Exercício 5: Usando setTimeout para Executar Código Após um Tempo
  • Exercício 6: Usando setInterval para Executar Código Repetidamente
  • Exercício 7: Acessando o Histórico do Navegador
  • Exercício 8: Obtendo Informações sobre o Navegador do Usuário
  • Exercício 9: Exibindo um Prompt para o Usuário
  • Exercício 10: Exibindo uma Caixa de Confirmação


Obs: Após o término dos exercícios, volte para o capítulo de 'Storage'.
