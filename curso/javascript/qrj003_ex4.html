Exercício 4: Tomando Decisões

  • Descrição: Crie código javascript que peça para o usuário digitar sua idade usando a função prompt(). Com base na idade, utilize uma estrutura if...else para exibir uma mensagem informando se ele é maior ou menor de idade.
  • Conceitos Teóricos:
    • A função global prompt() para receber entrada do usuário.
    • Estruturas condicionais: if, else.
    • Operadores de comparação: >, <.
    • Conversão de tipo (parseInt()) para garantir que a entrada seja tratada como número.
