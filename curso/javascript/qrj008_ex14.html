Exercício 14: Capturando o Valor de um Input

  • Tarefa: Crie um campo de texto (<input type="text">) no seu HTML com o ID "meuInput" e um botão com o ID "mostrarValor". Adicione um "event listener" de "click" ao botão. Quando o botão for clicado, capture o valor digitado no campo de texto e exiba-o em um alert().
  • Conceitos Teóricos:
    • O evento de "click" em botões.
    • A propriedade element.value para acessar ou modificar o valor de elementos de formulário (como <input>, <textarea>, <select>).
