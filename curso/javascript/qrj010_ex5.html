Exercício 5: Funções que Realizam Operações em Strings

  • Descrição: Defina uma função chamada primeiraLetra que aceite uma string como parâmetro e retorne a primeira letra dessa string. Chame a função com uma palavra e imprima a letra retornada.
  • Conceitos Teóricos:
    • Como trabalhar com strings dentro de funções.
    • Acessando caracteres de uma string por índice (como em arrays).
