Exercício 10: Persistindo um Pequeno Formulário com localStorage

  • Tarefa: Crie um formulário simples com campos de nome e email que:
    • Salve os dados do formulário no localStorage quando o usuário clicar em enviar.
    • Recupere e preencha os campos com os dados salvos quando a página for carregada.
    • Mostre uma mensagem de sucesso após salvar os dados.
  • Conceitos Teóricos:
    • Como usar localStorage para persistir dados de formulários.
    • Como recuperar e preencher campos de formulário automaticamente.
    • Como combinar localStorage com eventos de formulário (submit).
