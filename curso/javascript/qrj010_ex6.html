Exercício 6: Funções que Verificam Condições

  • Descrição: Defina uma função chamada ehPar que aceite um número como parâmetro e retorne true se o número for par e false se for ímpar. Chame a função com diferentes números e imprima o valor booleano retornado.
  • Conceitos Teóricos:
    • Usando estruturas condicionais (if...else) dentro de funções.
    • Funções que retornam valores booleanos com base em uma condição.
    • O operador módulo (%) para verificar o resto de uma divisão.
