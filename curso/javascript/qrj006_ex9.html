Exercício 9: Criando uma Cópia de um Array (Slice)

  • Descrição: Use o array de frutas dos exercícios anteriores. Crie uma cópia do array usando o método slice(). Modifique a cópia (adicione ou remova elementos) e imprima tanto o array original quanto a cópia modificada no console.
  • Conceitos Teóricos:
    • O método slice() para criar uma cópia superficial de um array.
    • Como slice() não modifica o array original.
    • Parâmetros opcionais do método slice() para selecionar um intervalo específico.
