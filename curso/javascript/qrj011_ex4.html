Exercício 4: Abrindo uma Nova Janela

  • Descrição: Crie um botão no seu HTML. Adicione um "event listener" de "click" a este botão. Quando o botão for clicado, use o método window.open() para abrir uma nova janela ou aba do navegador com um URL específico.
  • Conceitos Teóricos:
    • O método window.open('url', 'nomeDaJanela', 'especificacoes') para abrir novas janelas.
    • Os parâmetros do método window.open() (URL, nome da janela, especificações como largura e altura).
