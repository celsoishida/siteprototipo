Exercício 3: Funções com Retorno de Valor

  • Descrição: Defina uma função chamada dobrar que aceite um parâmetro numérico. Dentro da função, retorne o dobro desse número. Chame a função com um número e imprima o valor retornado no console.
  • Conceitos Teóricos:
    • O uso da palavra-chave return para retornar um valor de uma função.
    • Como o valor retornado pode ser usado em outras partes do código (atribuído a uma variável, impresso, etc.).
