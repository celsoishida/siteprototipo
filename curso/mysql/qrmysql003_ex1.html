Exercício 1: Quantas contrações e demissões teve em Curitiba?

  • Descrição: Utilizando a tabela MovCuritiba, descubra quantas contrações (tipomovimentacao referente a contratação) e quantas demissões (tipomovimentacao referente a demissão) ocorreram em Curitiba.
  • Conteúdo Teórico:
    • O comando SELECT é utilizado para consultar dados em tabelas do banco de dados.
    • Você pode utilizar funções de agregação como COUNT e SUM para obter totais.
    • Exemplo de estrutura da tabela utilizada:

CREATE TABLE MovCuritiba (
    competenciamov INT,
    secao VARCHAR(10),
    subclasse INT,
    saldomovimentacao INT,
    cbo2002ocupacao INT,
    categoria INT,
    graudeinstrucao INT,
    idade INT null,
    horascontratuais DECIMAL(5,2),
    racacor INT,
    sexo INT,
    tipoempregador INT,
    tipoestabelecimento INT,
    tipomovimentacao INT,
    tipodedeficiencia INT,
    indtrabintermitente INT,
    indtrabparcial INT,
    salario DECIMAL(10,2), 
    tamestabjan INT,
    indicadoraprendiz INT,
    origemdainformacao INT,
    competenciadec INT,
    indicadordeforadoprazo INT,
    unidadesalariocodigo INT,
    valorsalariofixo DECIMAL(10,2)
);


Dica:
Para contar as contratações e demissões, utilize SELECT COUNT(1) filtrando pelo campo saldomovimentacao (por exemplo, 1 para contratação e -1 para demissão).


Sugestão de Consulta:
SELECT tipomovimentacao, COUNT(*) AS total
FROM MovCuritiba
GROUP BY tipomovimentacao;

Dica:
Utilize o campo tipomovimentacao para diferenciar contratações e demissões.
