Exercício 5: Algum cientista de dados foi contratado? Qual foi seu salário?

  • Descrição: Utilizando a tabela MovCuritiba, descubra se algum cientista de dados foi contratado e, em caso afirmativo, qual foi o salário dessa contratação. Utilize o campo cbo2002ocupacao correspondente à profissão de cientista de dados.
  • Conteúdo Teórico:
    • O comando SELECT é utilizado para consultar dados em tabelas do banco de dados.
    • Você pode utilizar WHERE para filtrar registros por profissão e tipo de movimentação.
    • Exemplo de estrutura da tabela utilizada:

CREATE TABLE MovCuritiba (
    competenciamov INT,
    secao VARCHAR(10),
    subclasse INT,
    saldomovimentacao INT,
    cbo2002ocupacao INT,
    categoria INT,
    graudeinstrucao INT,
    idade INT null,
    horascontratuais DECIMAL(5,2),
    racacor INT,
    sexo INT,
    tipoempregador INT,
    tipoestabelecimento INT,
    tipomovimentacao INT,
    tipodedeficiencia INT,
    indtrabintermitente INT,
    indtrabparcial INT,
    salario DECIMAL(10,2), 
    tamestabjan INT,
    indicadoraprendiz INT,
    origemdainformacao INT,
    competenciadec INT,
    indicadordeforadoprazo INT,
    unidadesalariocodigo INT,
    valorsalariofixo DECIMAL(10,2)
);


Dica:
Consulte o código CBO da profissão de cientista de dados (por exemplo, 212415) e filtre por tipomovimentacao = 1 (contratação).


Sugestão de Consulta:
SELECT salario
FROM MovCuritiba
WHERE cbo2002ocupacao = 212415 AND tipomovimentacao = 1;

Dica:
Use o código CBO correto para cientista de dados e filtre por contratações.
