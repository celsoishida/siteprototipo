Exercício 8: Deletando um Cliente Específico (DELETE FROM - Exclusão Simples)

Descrição:
O cliente 'Fernanda Oliveira' (ID 3) solicitou a exclusão de seus dados.
Deletar o registro do cliente cujo id é 3.

Conteúdo Teórico:
A instrução DELETE FROM é usada para remover uma ou mais linhas (registros) de uma tabela. Assim como UPDATE, a cláusula WHERE é fundamental para especificar quais registros serão deletados. Muito cuidado: DELETE FROM nome_da_tabela; sem WHERE deleta todas as linhas da tabela!
SQL
DELETE FROM nome_da_tabela
WHERE alguma_condicao;
