G4 Subquery

Exercício 6: Todos os Nomes (Sem Duplicatas)

Descrição: Crie uma lista com todos os nomes de clientes e todos os nomes de autores, mas sem duplicar os nomes que aparecem em ambas as tabelas.

Conteúdo Teórico:
Este exercício foca no operador UNION. A principal diferença para o UNION ALL é que o UNION automaticamente remove as linhas duplicadas do resultado final, garantindo uma lista de valores únicos.
