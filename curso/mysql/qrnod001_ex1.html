Exercício 1: Nosso Primeiro Servidor Express

  • Descrição: Crie um servidor Node.js usando o Express.js que responda com a mensagem "Olá, Express!" quando acessado na rota raiz (/).
  • Conteúdo Teórico:
    • Express.js é um framework web minimalista e flexível para Node.js que simplifica a criação de aplicações web e APIs. Ele oferece ferramentas para roteamento, gerenciamento de requisições e respostas, e muito mais.
    • Para começar, você precisa instalar o Express (npm install express) e depois importá-lo no seu arquivo JavaScript.
    • express() cria uma instância do aplicativo Express.
    • app.listen(port, callback) inicia o servidor e o faz "escutar" em uma porta específica.

Exemplo de Código:

const express = require('express');
const app = express();
const port = 3000;
app.get('/', (req, res) => {
  res.send('Olá, Express!');
});
app.listen(port, () => {
  console.log(`Servidor rodando em http://localhost:${port}`);
});



Dica:
Lembre-se de criar um arquivo package.json usando npm init -y antes de instalar o Express.


Sugestão de Extensão:
Altere a porta do servidor para 5000.
