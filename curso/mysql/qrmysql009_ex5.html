Exercício 5: Atualizando a Nacionalidade de um Autor (UPDATE - Modificação de um Registro)

Descrição:
'José Saramago' é um autor de nacionalidade 'Portuguesa'. Por um erro de digitação, a nacionalidade dele foi registrada incorretamente.
Atualize a nacionalidade de 'José Saramago' para 'Luso'.

Conteúdo Teórico:
A instrução UPDATE é utilizada para modificar dados existentes em uma tabela. A cláusula SET é usada para especificar quais colunas serão atualizadas e seus novos valores. A cláusula WHERE é crucial para filtrar quais registros serão afetados; sem ela, todos os registros da tabela seriam atualizados!
SQL
UPDATE nome_da_tabela
SET coluna_a_atualizar = novo_valor
WHERE alguma_condicao;
