Exercício 4: Existem informações de quantas profissões diferentes?

  • Descrição: Utilizando a tabela MovCuritiba, descubra quantas profissões diferentes existem registradas (campo cbo2002ocupacao).
  • Conteúdo Teórico:
    • O comando SELECT é utilizado para consultar dados em tabelas do banco de dados.
    • Você pode utilizar COUNT(DISTINCT ...) para contar valores únicos em uma coluna.
    • Exemplo de estrutura da tabela utilizada:

CREATE TABLE MovCuritiba (
    competenciamov INT,
    secao VARCHAR(10),
    subclasse INT,
    saldomovimentacao INT,
    cbo2002ocupacao INT,
    categoria INT,
    graudeinstrucao INT,
    idade INT null,
    horascontratuais DECIMAL(5,2),
    racacor INT,
    sexo INT,
    tipoempregador INT,
    tipoestabelecimento INT,
    tipomovimentacao INT,
    tipodedeficiencia INT,
    indtrabintermitente INT,
    indtrabparcial INT,
    salario DECIMAL(10,2), 
    tamestabjan INT,
    indicadoraprendiz INT,
    origemdainformacao INT,
    competenciadec INT,
    indicadordeforadoprazo INT,
    unidadesalariocodigo INT,
    valorsalariofixo DECIMAL(10,2)
);


Dica:
Use COUNT(DISTINCT cbo2002ocupacao) para saber quantas profissões diferentes existem.


Sugestão de Consulta:
SELECT COUNT(DISTINCT cbo2002ocupacao) AS total_profissoes
FROM MovCuritiba;
