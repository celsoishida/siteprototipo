G5 CRUD

Exercício 1: Inserindo um Novo Autor (INSERT INTO - Registro Simples)

Descrição:
Insira um novo autor na tabela autores com as seguintes informações:
Nome: 'Clarisse Falcão'
Nacionalidade: 'Brasileira'
Data de Nascimento: '1989-07-02'

Conteúdo Teórico:
A instrução INSERT INTO é usada para adicionar novas linhas (registros) a uma tabela. A forma mais simples de uso exige que você especifique os nomes das colunas e os valores correspondentes que serão inseridos. Se você fornecer valores para todas as colunas da tabela na ordem correta, pode omitir a lista de colunas.
SQL
INSERT INTO nome_da_tabela (coluna1, coluna2, ...)
VALUES (valor1, valor2, ...);
Ou, se for inserir em todas as colunas e na ordem correta:
INSERT INTO nome_da_tabela
VALUES (valor1, valor2, ...);
