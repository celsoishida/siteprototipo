Exercício 4: Inserindo Múltiplos Clientes (INSERT INTO - Múltiplos Registros)

Descrição:
Adicione dois novos clientes à tabela Cliente em uma única instrução.
Cliente 1: Nome: 'Paula Fernandes', Email: 'paula.fernandes@email.com'
Cliente 2: Nome: 'Ricardo Costa', Email: 'ricardo.costa@email.com'

Conteúdo Teórico:
O MySQL permite inserir várias linhas em uma única instrução INSERT INTO. Isso é feito fornecendo múltiplos conjuntos de valores, separados por vírgulas, após a cláusula VALUES. É uma forma eficiente de adicionar muitos registros de uma vez.
SQL
INSERT INTO nome_da_tabela (coluna1, coluna2)
VALUES
(valor1a, valor2a),
(valor1b, valor2b),
(valor1c, valor2c);
