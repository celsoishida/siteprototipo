G2 - JOIN

Olá! Vamos praticar comandos JOIN em MySQL. Tente resolver os exercícios abaixo utilizando as tabelas Autores e Livros.


CREATE TABLE Autores (
    id INT PRIMARY KEY,
    nome VARCHAR(100),
    nacionalidade VARCHAR(50),
    data_nascimento DATE,
    data_falecimento DATE
);

CREATE TABLE Livros (
    id INT PRIMARY KEY,
    titulo VARCHAR(200),
    ano_publicacao INT,
    autor_id INT,
    FOREIGN KEY (autor_id) REFERENCES Autores(id)
);
