Exercício 3: Qual a média salarial das demissões?

  • Descrição: Utilizando a tabela MovCuritiba, descubra qual é a média salarial das demissões (filtrando apenas os registros de demissão).
  • Conteúdo Teórico:
    • O comando SELECT é utilizado para consultar dados em tabelas do banco de dados.
    • Funções de agregação como AVG permitem calcular médias em colunas numéricas.
    • Exemplo de estrutura da tabela utilizada:

CREATE TABLE MovCuritiba (
    competenciamov INT,
    secao VARCHAR(10),
    subclasse INT,
    saldomovimentacao INT,
    cbo2002ocupacao INT,
    categoria INT,
    graudeinstrucao INT,
    idade INT null,
    horascontratuais DECIMAL(5,2),
    racacor INT,
    sexo INT,
    tipoempregador INT,
    tipoestabelecimento INT,
    tipomovimentacao INT,
    tipodedeficiencia INT,
    indtrabintermitente INT,
    indtrabparcial INT,
    salario DECIMAL(10,2), 
    tamestabjan INT,
    indicadoraprendiz INT,
    origemdainformacao INT,
    competenciadec INT,
    indicadordeforadoprazo INT,
    unidadesalariocodigo INT,
    valorsalariofixo DECIMAL(10,2)
);


Dica:
Utilize AVG(salario) e filtre apenas os registros de demissão (por exemplo, tipomovimentacao = 2 se 2 for o código para demissão).


Sugestão de Consulta:
SELECT AVG(salario) AS media_salarial_demissoes
FROM MovCuritiba
WHERE saldomovimentacao = -1;

Dica:
Use a função AVG para calcular a média e filtre pelo campo tipomovimentacao para considerar apenas demissões.
