Exercício 6: Selecione a idade e o salário de todas as contratações de programadores

  • Descrição: Utilizando a tabela MovCuritiba, selecione a idade e o salário de todas as contratações de programadores (cbo2002ocupacao correspondente a programador, por exemplo, 317105).
  • Conteúdo Teórico:
    • O comando SELECT é utilizado para consultar dados em tabelas do banco de dados.
    • Você pode utilizar WHERE para filtrar registros por profissão e tipo de movimentação.
    • Exemplo de estrutura da tabela utilizada:

CREATE TABLE MovCuritiba (
    competenciamov INT,
    secao VARCHAR(10),
    subclasse INT,
    saldomovimentacao INT,
    cbo2002ocupacao INT,
    categoria INT,
    graudeinstrucao INT,
    idade INT null,
    horascontratuais DECIMAL(5,2),
    racacor INT,
    sexo INT,
    tipoempregador INT,
    tipoestabelecimento INT,
    tipomovimentacao INT,
    tipodedeficiencia INT,
    indtrabintermitente INT,
    indtrabparcial INT,
    salario DECIMAL(10,2), 
    tamestabjan INT,
    indicadoraprendiz INT,
    origemdainformacao INT,
    competenciadec INT,
    indicadordeforadoprazo INT,
    unidadesalariocodigo INT,
    valorsalariofixo DECIMAL(10,2)
);


Dica:
Consulte o código CBO da profissão de programador (por exemplo, 317105) e filtre por tipomovimentacao = 1 (contratação).


Sugestão de Consulta:
SELECT idade, salario
FROM MovCuritiba
WHERE cbo2002ocupacao = 317105 AND tipomovimentacao = 1;
