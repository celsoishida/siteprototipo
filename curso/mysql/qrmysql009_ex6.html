Exercício 6: Atualizando Título e ISBN de um Livro (UPDATE - Modificando Múltiplas Colunas)

Descrição:
O livro 'Dom Casmurro' teve seu título e ISBN atualizados devido a uma nova edição.
Atualize o título de 'Dom Casmurro' para 'Dom Casmurro - Edição Revisada'.
Atualize o isbn do mesmo livro para '9788500000000'.

Conteúdo Teórico:
Você pode atualizar várias colunas em uma única instrução UPDATE separando cada par coluna = valor com uma vírgula na cláusula SET. Isso permite que você faça múltiplas modificações em um ou mais registros de forma eficiente.
SQL
UPDATE nome_da_tabela
SET coluna1 = novo_valor1, coluna2 = novo_valor2
WHERE alguma_condicao;
