Exercício 4: Usando Query Parameters

  • Descrição: Crie uma rota /pesquisar que receba um termo de busca como um query parameter (?termo=valor). A resposta deve ser "Você pesquisou por: [valor do termo]". Se nenhum termo for fornecido, responda "Nenhum termo de pesquisa fornecido.".
  • Conteúdo Teórico:
    • Query parameters (parâmetros de consulta) são pares chave-valor anexados à URL após um ponto de interrogação (?). Eles são usados para filtrar, classificar ou passar dados adicionais em uma requisição GET.
    • Você pode acessá-los através de req.query. Por exemplo, para ?categoria=livros, req.query.categoria seria "livros".

Exemplo de Código:

app.get('/pesquisar', (req, res) => {
  const termo = req.query.termo;
  if (termo) {
    res.send(`Você pesquisou por: ${termo}`);
  } else {
    res.send('Nenhum termo de pesquisa fornecido.');
  }
});



Dica:
Verifique se o req.query.termo existe antes de usá-lo para evitar erros.


Sugestão de Extensão:
Permita múltiplos query parameters, por exemplo, /produtos?categoria=eletronicos&marca=samsung.
