Exercício 1: Preparando o Ambiente MySQL

  • Descrição: Configure um servidor MySQL localmente (usando XAMPP, Docker, ou instalação direta) e crie um banco de dados chamado api_nodejs_db. Em seguida, execute o script SQL para criar a tabela Usuario.
  • Conteúdo Teórico:
    • Antes de conectar o Node.js a um banco de dados, precisamos ter o banco de dados e as tabelas criadas. O MySQL é um sistema de gerenciamento de banco de dados relacional muito popular.
    • SQL (Structured Query Language) é a linguagem padrão para gerenciar dados em bancos de dados relacionais.
    • A instrução CREATE DATABASE é usada para criar um novo banco de dados.
    • A instrução CREATE TABLE é usada para criar uma nova tabela.
    • AUTO_INCREMENT: Garante que o Id seja automaticamente incrementado para cada novo registro.
    • PRIMARY KEY: Define uma coluna como chave primária, garantindo unicidade e otimizando buscas.
    • NOT NULL: Impede que a coluna tenha um valor nulo.
    • UNIQUE: Garante que todos os valores na coluna sejam diferentes.
    • NULL: Permite que a coluna tenha um valor nulo.
    • VARCHAR(n): Tipo de dado para armazenar strings de caracteres de comprimento variável, com um limite de n caracteres.
    • CHAR(n): Tipo de dado para armazenar strings de caracteres de comprimento fixo de n caracteres.
    • SMALLINT: Tipo de dado para números inteiros pequenos.
    • NUMERIC(p,s): Tipo de dado para números com precisão e escala definidas. No seu caso, NUMERIC(9) significa 9 dígitos no total.
    • DATE: Tipo de dado para armazenar datas no formato 'YYYY-MM-DD'.
    • TEXT: Tipo de dado para armazenar grandes blocos de texto.
Exemplo de Código (SQL para criar o banco e a tabela):

-- Crie o banco de dados
CREATE DATABASE IF NOT EXISTS api_nodejs_db;
-- Use o banco de dados recém-criado
USE api_nodejs_db;
-- Crie a tabela Usuario
CREATE TABLE IF NOT EXISTS Usuario (
Id INT AUTO_INCREMENT PRIMARY KEY,
Nome VARCHAR(200) NOT NULL,
Email VARCHAR(250) UNIQUE NOT NULL,
CPF CHAR(11) NULL,
DDD SMALLINT NULL,
Telefone NUMERIC(9) NULL,
DataNiver DATE NULL,
Obs TEXT null
);

Dica:
Você pode usar ferramentas como o MySQL Workbench, phpMyAdmin (se estiver com XAMPP) ou a linha de comando do MySQL para executar esses comandos SQL.
