Exercício 10: Servindo Arquivos Estáticos

  • Descrição: Configure o Express para servir arquivos estáticos de uma pasta chamada public. Crie um arquivo index.html dentro dessa pasta com o conteúdo "<h1>Bem-vindo ao meu site estático!</h1>". Em seguida, acesse http://localhost:3000/index.html no navegador.
  • Conteúdo Teórico:
    • O Express pode servir arquivos estáticos (HTML, CSS, JavaScript, imagens, etc.) usando o middleware express.static().
    • Você precisa especificar o diretório onde seus arquivos estáticos estão localizados.
    • Quando você acessa uma URL que corresponde a um arquivo dentro do diretório estático, o Express o servirá automaticamente.

Exemplo de Código:

const path = require('path'); // Módulo nativo do Node.js para lidar com caminhos de arquivo
// ... seu setup do Express ...
app.use(express.static(path.join(__dirname, 'public')));
// Não é necessário uma rota app.get('/') para servir o index.html,
// o middleware static fará isso automaticamente se a URL for /index.html



Dica:
Crie a pasta public na mesma raiz do seu arquivo .js do servidor.


Sugestão de Extensão:
