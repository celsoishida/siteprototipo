Exercício 2: Instalando o Pacote mysql2

  • Descrição: Crie um novo projeto Node.js em uma pasta vazia. Dentro desse projeto, instale o pacote mysql2.
  • Conteúdo Teórico:
    • Para que o Node.js possa "conversar" com o MySQL, precisamos de um pacote (ou biblioteca) que faça essa ponte. O mysql2 é uma excelente escolha por ser moderno e ter suporte a Promises, o que facilita muito o trabalho com operações assíncronas.
    • npm init -y: Comando para inicializar um novo projeto Node.js, criando um arquivo package.json com configurações padrão. O -y aceita todas as opções padrão.
    • npm install <nome-do-pacote>: Comando para instalar um pacote e adicioná-lo às dependências do seu projeto.
Exemplo de Código (Comandos no terminal):

Navegue até a pasta do seu projeto
cd /caminho/do/seu/projeto
Inicialize o projeto Node.js
npm init -y
Instale o pacote mysql2
npm install mysql2

Dica:
Após a instalação, verifique se o mysql2 aparece na seção "dependencies" do seu arquivo package.json.
Sugestão de Extensão:
Explore o package.json e entenda o que cada campo significa.
