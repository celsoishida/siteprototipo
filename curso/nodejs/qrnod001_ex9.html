Exercício 9: Lidando com Múltiplos Middlewares em uma Rota

  • Descrição: Crie uma rota /informacao-avancada que utilize dois middlewares antes de sua função de rota final. O primeiro middleware deve registrar "Primeiro middleware executado.", e o segundo deve registrar "Segundo middleware executado.". A função de rota final deve responder com "Informação avançada.".
  • Conteúdo Teórico:
    • Você pode passar múltiplas funções de middleware para um método de rota (como app.get()). Elas serão executadas em sequência.
    • Cada middleware precisa chamar next() para passar o controle para o próximo middleware ou para a função de rota final.

Exemplo de Código:

const primeiroMiddleware = (req, res, next) => {
  console.log('Primeiro middleware executado.');
  next();
};
const segundoMiddleware = (req, res, next) => {
  console.log('Segundo middleware executado.');
  next();
};
app.get('/informacao-avancada', primeiroMiddleware, segundoMiddleware, (req, res) => {
  res.send('Informação avançada.');
});



Dica:
Verifique a ordem dos logs no seu terminal para confirmar a sequência de execução dos middlewares.


Sugestão de Extensão:
Faça com que o primeiro middleware adicione uma propriedade ao objeto req (ex: req.dados = 'algum dado') e o segundo middleware use essa propriedade para fazer alguma lógica.
