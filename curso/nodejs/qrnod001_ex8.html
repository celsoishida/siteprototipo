Exercício 8: Rota para Erro 404 (Not Found)

  • Descrição: Configure uma rota "coringa" no final do seu arquivo de rotas que lide com todas as requisições para rotas não definidas. Esta rota deve retornar um status 404 e a mensagem "Rota não encontrada.".
  • Conteúdo Teórico:
    • No Express, a ordem das rotas importa. A primeira rota que corresponde à URL da requisição será executada.
    • Você pode usar app.use() sem um caminho específico ou com * como último app.get() para capturar todas as requisições que não foram tratadas por rotas anteriores.
    • Retornar um status 404 é uma prática padrão para indicar que o recurso solicitado não foi encontrado.

Exemplo de Código:

// Suas rotas normais aqui...
app.get('/', (req, res) => {
  res.send('Página inicial.');
});
// ... outras rotas ...
// Rota 404 - SEMPRE no final
app.use((req, res, next) => {
  res.status(404).send('Rota não encontrada.');
});



Dica:
Qualquer rota que você digitar no navegador que não esteja explicitamente definida em seu código deve cair nesta rota 404.


Sugestão de Extensão:
Em vez de apenas uma mensagem de texto, envie um objeto JSON com uma mensagem de erro e o status.
