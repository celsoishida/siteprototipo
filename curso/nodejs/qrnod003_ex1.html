Exercício 1: Listando Dados (SELECT)

  • Descrição: Crie uma API que retorne todos os registros de uma tabela chamada produtos no seu banco de dados MySQL. A API deve responder a uma requisição GET na rota /produtos.
  • Conceitos Teóricos:
    • Node.js e Express: O Express é um framework minimalista e flexível de Node.js que nos ajuda a construir servidores web e APIs de forma rápida. Ele lida com rotas, requisições e respostas.
    • mysql2: Uma biblioteca para Node.js que permite a comunicação com bancos de dados MySQL. É uma alternativa moderna e eficiente ao mysql.
    • Rotas GET: São usadas para requisitar dados de um servidor.
    • SQL SELECT: A instrução SQL para recuperar dados de uma ou mais tabelas.

Dicas:
Comece instalando as dependências: npm init -y e depois npm install express mysql2.
Crie uma tabela produtos no seu banco de dados com alguns dados de exemplo para testar.
Sugestão de Extensão:
Altere a rota para /itens e liste itens de uma nova tabela itens.
