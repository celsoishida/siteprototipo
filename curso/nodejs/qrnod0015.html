Exemplo update API using copilot

Passos gerais para informar ao copilot:

  • Criar aplicativo Node.js ou aproveitar um já criado
  • Criar uma API Node para executar o update...
  • gerenalizar para receber os valores como parâmetros
  • Criar arquivo de teste
  • customizar mensagem de sucesso

Depois, altere manualmente as informações de conexão com o banco de dados no arquivo db.js

Obs: Volte à leitura do livro.
