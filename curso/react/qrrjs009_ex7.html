Exercício 7: useEffect - Limpeza

  • Descrição: Crie um componente que simule um timer. Ele deve ter um useEffect que configura um setInterval para exibir "Tick!" no console a cada segundo. É crucial que, quando o componente for desmontado, o setInterval seja limpo para evitar vazamentos de memória.
  • Conceitos Teóricos:
    • Função de Limpeza do useEffect: A função retornada pelo useEffect é a função de "limpeza". Ela é executada antes da próxima execução do efeito ou quando o componente é desmontado.
    • Desmontagem: Quando um componente é removido da árvore do DOM.
