Exercício 4: Componente com Props (Propriedades)

  • Descrição: Modifique o componente funcional Saudacao do Exercício 1 para que ele aceite uma propriedade chamada nome. A mensagem <h1> deve agora exibir "Olá, [nome]! Sou um componente funcional." onde [nome] é o valor passado pela prop. Renderize o componente Saudacao duas vezes, uma passando "João" e outra passando "Maria" como nome.
  • Conteúdo Teórico:
    • Props (propriedades) são argumentos que você passa para os componentes. Elas são a principal forma de comunicação entre componentes em React, permitindo que um componente pai envie dados para um componente filho.
    • Em componentes funcionais, as props são recebidas como um objeto no primeiro argumento da função.
    • Em componentes de classe, as props são acessadas através de this.props.
    • As props são somente leitura, ou seja, um componente nunca deve modificar suas próprias props.

Dica:
Para acessar as props em um componente funcional, use props.nome dentro da função. Ao renderizar, passe a prop como um atributo: <Saudacao nome="João" />.
Sugestão de Extensão:
Adicione uma segunda prop chamada sobrenome ao componente Saudacao e exiba o nome completo.
