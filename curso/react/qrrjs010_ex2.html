Exercício 2: Criando suas Primeiras Rotas com BrowserRouter e Routes

  • Descrição: Configure o BrowserRouter no seu arquivo index.js e crie dois componentes simples (Home.js e About.js). Defina as rotas para esses componentes usando Routes e Route no seu App.js.
  • Conceitos Teóricos:
    • BrowserRouter: O roteador principal que utiliza a API de histórico do HTML5 para manter a interface de usuário em sincronia com a URL.
    • Routes: Um componente que agrupa e gerencia os Routes. É onde você define suas rotas.
    • Route: Define um caminho (path) e o componente que será renderizado quando esse caminho for acessado.
