Exercício 2: Uso de className em vez de class

  • Descrição: Crie um elemento <div> com uma classe de nome: container.
  • Conteúdo Teórico: No JSX, usamos className no lugar de class, pois class é uma palavra reservada em JavaScript.

Exemplo de Código:

const elemento = <div className="container">Conteúdo</div>;

Dica: Se tentar usar class no JSX, o código não funcionará.
Sugestão de Extensão: Adicione mais estilos usando CSS externo.
