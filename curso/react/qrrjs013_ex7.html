Exercício 7: Requisições POST para Criar Recursos

  • Descrição: Crie um formulário com campos para "Título" e "Corpo" de um novo post. Ao enviar o formulário, o componente deve fazer uma requisição POST para a API JSONPlaceholder para criar um novo post e exibir a resposta (o post recém-criado) na tela.
    Endpoint: https://jsonplaceholder.typicode.com/posts
  • Conteúdo Teórico:
    • Requisições POST são usadas para enviar dados ao servidor e criar novos recursos.
    • No fetch, você precisa especificar o método 'POST', adicionar headers como 'Content-Type': 'application/json', e enviar os dados no body da requisição como uma string JSON (JSON.stringify(data)).
    • Com axios, o método axios.post() simplifica isso, você passa o URL e o objeto de dados diretamente.


Dica:
Lembre-se de prevenir o comportamento padrão de recarregamento do formulário com event.preventDefault().
Sugestão de Extensão:
Adicione feedback visual para o usuário, como uma mensagem de sucesso ou erro após a tentativa de criação do post.
