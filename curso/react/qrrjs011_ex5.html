Exercício 5: Múltiplos Contextos - Integrando Tema e Autenticação

  • Descrição: Combine os conceitos dos exercícios anteriores. Sua aplicação deve ter o ThemeContext e o AuthContext ativos simultaneamente. O Header deve exibir tanto o tema atual quanto o nome do usuário logado (se houver).
  • Conceitos Teóricos:
    • Você pode ter vários contextos em uma aplicação React. Basta envolver os componentes filhos com múltiplos Providers.
    • A ordem dos Providers geralmente não importa, a menos que um contexto dependa do valor de outro.

Exemplo de Código:

// ThemeContext.js, AuthContext.js (os mesmos dos exercícios anteriores)

// Header.js
import React, { useContext } from 'react';
import { ThemeContext } from './ThemeContext';
import { AuthContext } from './AuthContext';

function Header() {
  const { theme } = useContext(ThemeContext);
  const { user } = useContext(AuthContext);
  
  return (
    <header style={{ 
      background: theme === 'dark' ? '#333' : '#eee', 
      color: theme === 'dark' ? '#eee' : '#333',
      padding: '1rem',
      marginBottom: '1rem'
    }}>
      <h1>Minha Aplicação</h1>
      <p>Tema: {theme}</p>
      {user ? <p>Logado como: {user.name}</p> : <p>Não Logado</p>}
    </header>
  );
}

// App.js
import React from 'react';
import { ThemeProvider } from './ThemeContext';
import { AuthProvider } from './AuthContext';
import Header from './Header';
import ThemeSwitcher from './ThemeSwitcher';
import AuthButtons from './AuthButtons';

function App() {
  return (
    <ThemeProvider>
      <AuthProvider>
        <Header />
        <main style={{ padding: '1rem' }}>
          <h2>Conteúdo Principal</h2>
          <ThemeSwitcher />
          <AuthButtons />
        </main>
      </AuthProvider>
    </ThemeProvider>
  );
}



Dica:
Aninhe os Providers na ordem que fizer mais sentido para a sua aplicação. No caso, a ordem não afeta a funcionalidade.

Sugestão de Extensão:
Crie um componente de Dashboard que só seja renderizado se o usuário estiver logado e exiba informações personalizadas com base no tema.
