Exercício 1: Olá, Componente Funcional!

  • Descrição: Crie seu primeiro componente funcional chamado Saudacao que renderize um elemento <h1> com a mensagem "Olá, React! Sou um componente funcional."
  • Conteúdo Teórico:
    • Componentes funcionais são funções JavaScript simples que retornam JSX (JavaScript XML), uma sintaxe que nos permite escrever HTML dentro do JavaScript. Eles são a forma mais comum e moderna de criar componentes em React.
    • Para definir um componente funcional, você simplesmente escreve uma função JavaScript que começa com letra maiúscula (uma convenção importante em React).
    • O JSX retornado por um componente funcional é o que será exibido na interface do usuário.

Dica:
Lembre-se de que os componentes React devem começar com letra maiúscula (PascalCase).
Sugestão de Extensão:
Adicione um parágrafo <p> abaixo do <h1> com uma mensagem "Estou aprendendo muito!".
