Exercício 8: Manipulação de Dados da Resposta: Filtragem

  • Descrição: Busque uma lista de "todos" da API JSONPlaceholder (todos os todos). Exiba apenas os "todos" que estão completed: true.
    Endpoint: https://jsonplaceholder.typicode.com/todos
  • Conteúdo Teórico:
    • Após receber os dados da API, você pode usar métodos JavaScript como filter() para manipular e exibir apenas os dados relevantes.


Dica:
Use todo.completed === true ou simplesmente todo.completed para verificar se a propriedade é verdadeira.
Sugestão de Extensão:
Adicione um botão para alternar entre "Todos Completos" e "Todos Pendentes".
