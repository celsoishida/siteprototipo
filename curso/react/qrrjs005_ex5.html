Exercício 5: Múltiplas Props e Tipos de Dados

  • Descrição: Crie um componente funcional Produto que aceita as props nome, preco e disponivel (um booleano). O componente deve exibir o nome do produto em um <h3>, o preço em um <p> e, se disponivel for true, um <span> verde com "Em estoque", caso contrário, um <span> vermelho com "Esgotado". Renderize dois produtos: um disponível e outro esgotado.
  • Conteúdo Teórico:
    • Props podem ser de qualquer tipo de dado JavaScript: strings, números, booleanos, arrays, objetos e até mesmo funções.
    • Para passar valores que não são strings (como números ou booleanos), você deve envolvê-los em chaves {} no JSX.

Dica:
Use um operador ternário (condicao ? valorSeTrue : valorSeFalse) ou um if para renderização condicional baseado na prop disponivel.
Sugestão de Extensão:
Adicione uma prop quantidade e exiba a quantidade apenas se o produto estiver disponível.
