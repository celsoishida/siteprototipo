Exercício 4: Requisições com axios

  • Descrição: Refatore o componente do Exercício 2 (lista de posts) para usar a biblioteca axios em vez de fetch.
    Endpoint: https://jsonplaceholder.typicode.com/posts?_limit=5
  • Conteúdo Teórico:
    • axios é uma biblioteca popular baseada em Promises para fazer requisições HTTP. Oferece uma API mais concisa e funcionalidades adicionais como interceptors e tratamento automático de JSON.
    • Para usá-lo, primeiro instale-o no seu projeto: npm install axios ou yarn add axios.
    • As requisições axios retornam uma Promise, assim como fetch, mas os dados JSON já vêm na propriedade data do objeto de resposta.


Dica:
Lembre-se de importar axios no seu componente.
Sugestão de Extensão:
Adicione um botão "Recarregar Posts" que faça uma nova requisição.
