Exercício 2: Exibindo uma Lista de Recursos

  • Descrição: Crie um componente que busque uma lista de "posts" da API JSONPlaceholder e os exiba como uma lista não ordenada (<ul>), onde cada item (<li>) mostra o título de um post.
    Endpoint: https://jsonplaceholder.typicode.com/posts?_limit=5
  • Conteúdo Teórico:
    • Ao buscar uma coleção de dados, você geralmente os armazena em um estado como um array.
    • Para renderizar uma lista de elementos em React, você pode usar o método map() em um array de dados, retornando um elemento JSX para cada item.
    • É crucial fornecer uma key única para cada item renderizado em uma lista. A key ajuda o React a identificar quais itens foram alterados, adicionados ou removidos.


Dica:
Use o id de cada post como key.
