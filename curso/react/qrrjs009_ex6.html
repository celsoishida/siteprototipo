Exercício 6: useEffect - Atualização do Estado

  • Descrição: Retome o componente do Exercício 1 (Contador). Agora, utilize useEffect para que, a cada vez que o valor do contador for atualizado, uma mensagem no console exiba o novo valor do contador.
  • Conceitos Teóricos:
    • useEffect com Dependências: Quando o useEffect possui um array de dependências com variáveis, a função do efeito será executada sempre que uma dessas variáveis mudar de valor (além da montagem inicial).

Dicas:
Inclua a variável contador no array de dependências do useEffect.
Ex: useEffect(() => { /* ... */ }, [contador]);
