Exercício 8: Usando NavLink para Estilizar Links Ativos

  • Descrição: Substitua os Links do seu menu de navegação por NavLink e utilize a prop className para aplicar um estilo diferente ao link da página ativa.
  • Conceitos Teóricos:
    • NavLink: Um componente especial do react-router-dom que é como um Link, mas com a funcionalidade adicional de aplicar uma classe CSS quando o link está "ativo" (ou seja, a URL atual corresponde ao to do NavLink).
    • className: Uma prop que pode receber uma string ou uma função para aplicar classes CSS dinamicamente.

Exemplo de Código:

// src/App.js (atualizado)
import React from 'react';
import { Routes, Route, NavLink } from 'react-router-dom'; // Importe NavLink
import Home from './components/Home';
import About from './components/About';
import Dashboard from './components/Dashboard';
import Profile from './components/Profile';
import Settings from './components/Settings';
import Products from './components/Products';
import ProductDetail from './components/ProductDetail';
import NotFound from './components/NotFound';
import './App.css'; // Crie este arquivo CSS

function App() {
  const activeLinkStyle = ({ isActive }) => ({
    fontWeight: isActive ? 'bold' : 'normal',
    color: isActive ? 'red' : 'blue',
  });

  return (
    <div>
      <h1>Meu App de Rotas</h1>
      <nav>
        <ul>
          <li>
            <NavLink to="/" style={activeLinkStyle}>Home</NavLink>
          </li>
          <li>
            <NavLink to="/about" style={activeLinkStyle}>Sobre</NavLink>
          </li>
          <li>
            <NavLink to="/dashboard" style={activeLinkStyle}>Dashboard</NavLink>
          </li>
          <li>
            <NavLink to="/products" style={activeLinkStyle}>Produtos</NavLink>
          </li>
        </ul>
      </nav>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
        <Route path="/dashboard" element={<Dashboard />}>
          <Route path="profile" element={<Profile />} />
          <Route path="settings" element={<Settings />} />
        </Route>
        <Route path="/products" element={<Products />} />
        <Route path="/products/:id" element={<ProductDetail />} />
        <Route path="*" element={<NotFound />} />
      </Routes>
    </div>
  );
}

export default App;



Dicas:
- A prop `style` no `NavLink` pode receber uma função que recebe um objeto com `isActive` para estilização condicional.
- Você também pode usar a prop `className` que recebe uma função com `{ isActive }` para aplicar classes CSS.

Sugestão de Extensão:
- Experimente usar a prop `end` no `NavLink` para rotas que você não quer que sejam ativadas por sub-rotas (ex: se `/` for ativo para `/dashboard`).
