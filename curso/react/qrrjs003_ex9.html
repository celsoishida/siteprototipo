Exercício 9. Renderização Condicional Simples

  • Descrição: Exiba um elemento apenas se uma variável for verdadeira.
  • Conteúdo Teórico: Podemos controlar o que será renderizado usando condicionais.
  • Exemplo de Código: 
    
const mostrar = true;
const elemento = mostrar ? <p>Visível!</p> : null;
  • Dica: O operador ternário ajuda a escrever lógica simples dentro de JSX.
  • Sugestão de Extensão: Exiba um texto alternativo quando mostrar for false.

Dicas:
  Use o operador ternário para decidir o que renderizar.
  
    const mostrar = true;
    const elemento = mostrar ? <p>Visível!</p> : null;
  
Sugestão de Extensão:
  Exiba um texto alternativo quando mostrar for false:
  
    const elemento = mostrar ? <p>Visível!</p> : <p>Oculto!</p>;
