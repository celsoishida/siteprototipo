Exercício 8: Gerenciando uma Lista de Tarefas com Redux

  • Descrição: Crie um novo slice Redux para gerenciar uma lista de tarefas (to-dos). As tarefas devem ter um ID, texto e status de conclusão (completed). Implemente ações para adicionar uma tarefa, alternar o status de conclusão de uma tarefa e remover uma tarefa. Crie componentes para exibir a lista e adicionar novas tarefas.
  • Conceitos Teóricos:
    • Múltiplos Slices: Em aplicações Redux maiores, é comum dividir o estado em fatias ("slices") lógicas, cada uma com seu próprio reducer e ações relacionadas. O Redux Toolkit facilita isso.
    • Imutabilidade: Em Redux, você nunca deve modificar o estado diretamente. Em vez disso, retorne sempre um novo objeto de estado com as alterações desejadas. O Redux Toolkit (com Immer.js) permite que você escreva código "mutante" dentro dos reducers de createSlice, e ele se encarrega de criar cópias imutáveis para você.

Exemplo de Código:

// store/index.js
import { configureStore, createSlice } from '@reduxjs/toolkit';

// Slice do Contador (do exercício anterior, apenas para referência)
const counterSlice = createSlice({
  name: 'counter',
  initialState: { value: 0 },
  reducers: {
    increment: (state) => { state.value += 1; },
    decrement: (state) => { state.value -= 1; },
  },
});

export const { increment, decrement } = counterSlice.actions;

// Novo Slice de Tarefas
const todosSlice = createSlice({
  name: 'todos',
  initialState: [], // Array de objetos { id, text, completed }
  reducers: {
    addTodo: (state, action) => {
      state.push({ id: Date.now(), text: action.payload, completed: false });
    },
    toggleTodo: (state, action) => {
      const todo = state.find((todo) => todo.id === action.payload);
      if (todo) {
        todo.completed = !todo.completed;
      }
    },
    removeTodo: (state, action) => {
      return state.filter((todo) => todo.id !== action.payload);
    },
  },
});

export const { addTodo, toggleTodo, removeTodo } = todosSlice.actions;

export const store = configureStore({
  reducer: {
    counter: counterSlice.reducer,
    todos: todosSlice.reducer, // Adicione o novo reducer
  },
});

// TodoList.js
import React from 'react';
import { useSelector, useDispatch } from 'react-redux';
import { toggleTodo, removeTodo } from './store';

function TodoList() {
  const todos = useSelector((state) => state.todos);
  const dispatch = useDispatch();

  return (
    <ul style={{ listStyle: 'none', padding: 0 }}>
      {todos.map((todo) => (
        <li key={todo.id} style={{ 
          textDecoration: todo.completed ? 'line-through' : 'none',
          margin: '8px 0',
          padding: '8px',
          backgroundColor: todo.completed ? '#f0f0f0' : 'white',
          borderRadius: '4px',
          display: 'flex',
          justifyContent: 'space-between',
          alignItems: 'center'
        }}>
          <span 
            onClick={() => dispatch(toggleTodo(todo.id))}
            style={{ cursor: 'pointer', flexGrow: 1 }}
          >
            {todo.text}
          </span>
          <button 
            onClick={(e) => {
              e.stopPropagation();
              dispatch(removeTodo(todo.id));
            }}
            style={{
              marginLeft: '10px',
              backgroundColor: '#ff6b6b',
              color: 'white',
              border: 'none',
              borderRadius: '4px',
              padding: '4px 8px',
              cursor: 'pointer'
            }}
          >
            Remover
          </button>
        </li>
      ))}
    </ul>
  );
}

// AddTodoForm.js
import React, { useState } from 'react';
import { useDispatch } from 'react-redux';
import { addTodo } from './store';

function AddTodoForm() {
  const [todoText, setTodoText] = useState('');
  const dispatch = useDispatch();

  const handleSubmit = (e) => {
    e.preventDefault();
    if (todoText.trim()) {
      dispatch(addTodo(todoText));
      setTodoText('');
    }
  };

  return (
    <form onSubmit={handleSubmit} style={{ marginBottom: '20px' }}>
      <input
        type="text"
        value={todoText}
        onChange={(e) => setTodoText(e.target.value)}
        placeholder="Adicionar nova tarefa"
        style={{
          padding: '8px',
          marginRight: '8px',
          borderRadius: '4px',
          border: '1px solid #ccc',
          width: '300px',
          maxWidth: '100%'
        }}
      />
      <button 
        type="submit"
        style={{
          padding: '8px 16px',
          backgroundColor: '#4caf50',
          color: 'white',
          border: 'none',
          borderRadius: '4px',
          cursor: 'pointer'
        }}
      >
        Adicionar
      </button>
    </form>
  );
}

// App.js
import React from 'react';
import TodoList from './TodoList';
import AddTodoForm from './AddTodoForm';

function App() {
  return (
    <div style={{ maxWidth: '600px', margin: '0 auto', padding: '20px', fontFamily: 'Arial, sans-serif' }}>
      <h1 style={{ color: '#333' }}>Minhas Tarefas com Redux</h1>
      <AddTodoForm />
      <TodoList />
    </div>
  );
}

export default App;



Dica:
Ao adicionar um novo item, use Date.now() ou uma biblioteca como uuid para gerar um ID único.

Sugestão de Extensão:
Adicione filtros para exibir apenas tarefas concluídas, não concluídas ou todas as tarefas.
