Exercício 10: Redirecionamento com Navigate e index routes

  • Descrição: No seu `Dashboard.js`, adicione uma rota `index` que redirecione para `profile` automaticamente quando o usuário acessar apenas `/dashboard`.
  • Conceitos Teóricos:
    • `index` Route: Uma rota aninhada com a propriedade `index` definida como `true` será renderizada quando o caminho da rota pai for acessado, e nenhuma outra rota aninhada corresponder. É como um `default` para rotas aninhadas.
    • Navigate: Um componente que renderiza e automaticamente navega para um novo local. É um wrapper declarativo para `useNavigate`.

Exemplo de Código:

// src/App.js (atualizado)
import React from 'react';
import { Routes, Route, NavLink, Navigate } from 'react-router-dom'; // Importe Navigate
import Home from './components/Home';
import About from './components/About';
import Dashboard from './components/Dashboard';
import Profile from './components/Profile';
import Settings from './components/Settings';
import Products from './components/Products';
import ProductDetail from './components/ProductDetail';
import NotFound from './components/NotFound';
import './App.css';

function App() {
  const activeLinkStyle = ({ isActive }) => ({
    fontWeight: isActive ? 'bold' : 'normal',
    color: isActive ? 'red' : 'blue',
  });

  return (
    <div>
      <h1>Meu App de Rotas</h1>
      <nav>
        <ul>
          <li>
            <NavLink to="/" style={activeLinkStyle}>Home</NavLink>
          </li>
          <li>
            <NavLink to="/about" style={activeLinkStyle}>Sobre</NavLink>
          </li>
          <li>
            <NavLink to="/dashboard" style={activeLinkStyle}>Dashboard</NavLink>
          </li>
          <li>
            <NavLink to="/products" style={activeLinkStyle}>Produtos</NavLink>
          </li>
        </ul>
      </nav>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
        <Route path="/dashboard" element={<Dashboard />}>
          <Route index element={<Navigate to="profile" replace />} /> {/* Rota index com redirecionamento */}
          <Route path="profile" element={<Profile />} />
          <Route path="settings" element={<Settings />} />
        </Route>
        <Route path="/products" element={<Products />} />
        <Route path="/products/:id" element={<ProductDetail />} />
        <Route path="*" element={<NotFound />} />
      </Routes>
    </div>
  );
}

export default App;



Dicas:
- A prop `replace` no `Navigate` impede que a rota de destino seja adicionada ao histórico do navegador, substituindo a entrada atual.
- A rota `index` é útil para definir uma página padrão para um layout aninhado.

Sugestão de Extensão:
- Crie um componente de layout para o `Dashboard` que inclua uma barra lateral de navegação e use `Outlet` para renderizar o conteúdo.
