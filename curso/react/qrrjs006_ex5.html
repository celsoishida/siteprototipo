Exercício 5: Desestruturação de Props

  • Descrição: Refatore o componente PerfilUsuario (do Exercício 2) para utilizar a desestruturação de props.
  • Conceitos Teóricos:
    • A desestruturação de objetos é uma funcionalidade do JavaScript que permite extrair valores de propriedades de objetos para variáveis separadas. Isso torna o código mais conciso e legível, especialmente quando você tem muitas props.
    • Em vez de acessar props.nome e props.idade, você pode desestruturar o objeto props diretamente nos parâmetros da função do componente: function PerfilUsuario({ nome, idade }).

Exemplo de Código:

// App.js
import React from 'react';

// Componente PerfilUsuario refatorado com desestruturação
function PerfilUsuario({ nome, idade }) {
  return (
    <div>
      <p>Nome: {nome}</p>
      <p>Idade: {idade}</p>
    </div>
  );
}

function App() {
  return (
    <div>
      <PerfilUsuario nome="Roberto" idade={25} />
    </div>
  );
}

export default App;



Dicas:
A desestruturação pode parecer um pouco estranha no início, mas é uma prática muito comum e útil em React.
Sugestão de Extensão:
Adicione mais uma prop (por exemplo, email) e inclua-a na desestruturação e exibição.
