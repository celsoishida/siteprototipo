Exercício 1: Nosso Primeiro State!

  • Descrição: Crie um componente funcional jsx chamado Saudacao que exiba a mensagem "Olá, mundo!" usando um state para armazenar essa mensagem. O objetivo é ver o state em ação de forma simples.
  • Conteúdo Teórico:
    • State em React: É um objeto JavaScript que armazena dados que são específicos de um componente e que podem mudar ao longo do tempo. Quando o state de um componente muda, o componente é re-renderizado para refletir essas mudanças na interface do usuário.
    • useState: É um Hook do React que permite adicionar state a componentes funcionais. Ele retorna um par de valores: o estado atual e uma função para atualizá-lo.

Exemplo de Código:

// App.js
import React, { useState } from 'react';

function Saudacao() {
  const [mensagem, setMensagem] = useState('Olá, mundo!');

  return (
    <h1>{mensagem}</h1>
  );
}

export default Saudacao;



Dica:
Lembre-se de importar o useState do React!
Sugestão de Extensão:
Tente mudar a mensagem inicial do state e veja o que acontece.
