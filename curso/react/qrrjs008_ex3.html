Exercício 3 (Variação): Contador Visual de Emojis

  • Descrição: Crie um componente React que exiba uma quantidade de emojis de estrelas (⭐) equivalente a um número inicial (começando em 0). Adicione dois botões: um "Adicionar Estrela" que incrementa o número em 1 (e adiciona uma estrela), e um "Remover Estrela" que decrementa o número em 1 (e remove uma estrela), mas nunca deixando o contador abaixo de zero.
  • Conteúdo Teórico:
    • Imutabilidade do Estado em Ação: Para atualizar o nosso contador, sempre criamos um novo valor de estado, seja ele um número ou um array. Nunca modificamos o estado diretamente. Quando usamos setContador(contador + 1), não estamos alterando o contador existente, mas sim dizendo ao React qual será o próximo valor do contador.
    • Renderização de Listas e Mapeamento: Para exibir múltiplos emojis, vamos usar o método Array.from() ou um loop map em um array para gerar dinamicamente os elementos JSX. Isso nos permite "desenhar" a quantidade correta de estrelas com base no nosso estado numérico.

Dica:
Pense em como garantir que a contagem de estrelas nunca seja negativa. Você pode usar uma condição antes de decrementar o valor do estado.
Sugestão de Extensão:
- Adicione um botão "Resetar" que zere a contagem de estrelas.
- Implemente um limite máximo de estrelas (ex: 10 estrelas), desabilitando o botão "Adicionar Estrela" quando o limite for atingido.
- Permita que o usuário escolha um emoji diferente para contar, talvez com um input de texto para o emoji.
