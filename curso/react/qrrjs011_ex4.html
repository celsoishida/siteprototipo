Exercício 4: Compartilhando Estado de Usuário com Context API

  • Descrição: Crie um novo contexto chamado AuthContext para gerenciar o estado de autenticação de um usuário. O estado deve conter um objeto user (inicialmente null) e uma função login e logout. Crie componentes para simular login e exibir o nome do usuário logado.
  • Conceitos Teóricos:
    • A Context API pode ser usada para compartilhar qualquer tipo de dado, não apenas temas. Informações de autenticação, carrinho de compras, configurações de idioma são exemplos comuns.
    • Passar objetos com múltiplas propriedades (estado e funções) no value do Provider é uma prática comum para expor a interface de um contexto.

Exemplo de Código:

// AuthContext.js
import React, { createContext, useState } from 'react';

export const AuthContext = createContext();

export const AuthProvider = ({ children }) => {
  const [user, setUser] = useState(null);
  
  const login = (username) => {
    setUser({ name: username });
  };
  
  const logout = () => {
    setUser(null);
  };
  
  return (
    <AuthContext.Provider value={{ user, login, logout }}>
      {children}
    </AuthContext.Provider>
  );
};

// App.js
import React from 'react';
import { AuthProvider } from './AuthContext';
import UserProfile from './UserProfile';
import AuthButtons from './AuthButtons';

function App() {
  return (
    <AuthProvider>
      <h1>Aplicação de Autenticação</h1>
      <UserProfile />
      <AuthButtons />
    </AuthProvider>
  );
}

// UserProfile.js
import React, { useContext } from 'react';
import { AuthContext } from './AuthContext';

function UserProfile() {
  const { user } = useContext(AuthContext);
  
  return (
    <div>
      {user ? <p>Bem-vindo, {user.name}!</p> : <p>Nenhum usuário logado.</p>}
    </div>
  );
}

// AuthButtons.js
import React, { useContext } from 'react';
import { AuthContext } from './AuthContext';

function AuthButtons() {
  const { user, login, logout } = useContext(AuthContext);
  
  return (
    <div>
      {!user ? (
        <button onClick={() => login('João Silva')}>Login</button>
      ) : (
        <button onClick={logout}>Logout</button>
      )}
    </div>
  );
}



Dica:
Considere o value do Provider como um objeto que contém tanto os dados que você quer compartilhar quanto as funções para manipulá-los.

Sugestão de Extensão:
Adicione um formulário de login para que o usuário possa digitar o nome antes de fazer login.
