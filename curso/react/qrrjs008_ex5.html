Exercício 5: Entrada de Texto Dinâmica

  • Descrição: Crie um componente que contenha um campo de input de texto (<input type="text" />) e um parágrafo. O texto digitado no input deve ser exibido em tempo real no parágrafo.
  • Conteúdo Teórico:
    • Evento onChange: Para monitorar as mudanças em campos de formulário (como input, textarea, select), usamos o evento onChange. Este evento é disparado a cada vez que o valor do campo muda.
    • Acessando o Valor do Input: Dentro da função de manipulação do onChange, o evento (e) contém a propriedade e.target.value, que é o valor atual do campo de input.
    • Controlled Components: No React, é uma boa prática ter os elementos de formulário "controlados" pelo estado do componente. Isso significa que o valor do input é sempre derivado do estado.
