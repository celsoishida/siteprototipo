Exercício 8: Validação Simples de Formulário

  • Descrição: Crie um formulário simples com um campo de input de texto e um botão "Enviar". O botão deve estar desabilitado inicialmente. Ele só deve ser habilitado se o campo de input não estiver vazio (ou seja, se tiver pelo menos um caractere).
  • Conteúdo Teórico:
    • Propriedade disabled: A propriedade disabled em elementos de formulário (como button, input) pode ser definida como true ou false para controlar se o elemento está habilitado ou desabilitado.

Dica:
Use um estado para o valor do input e outro estado (ou uma variável derivada do estado do input) para controlar a propriedade disabled do botão.
Sugestão de Extensão:
Adicione uma validação de que o texto digitado deve ter no mínimo 5 caracteres para habilitar o botão.
