Exercício 10. Listas e Uso de .map()

  • Descrição: Renderize uma lista de itens com .map().
  • Conteúdo Teórico: O método .map() ajuda a transformar arrays em elementos JSX.
  • Exemplo de Código: 
    
const itens = ["React", "Vue", "Angular"];
const lista = itens.map((item) => <li key={item}>{item}</li>);
  • Dica: Sempre use key ao renderizar listas.
  • Sugestão de Extensão: Adicione números às listas dinamicamente.

Dicas:
  Use o método map para transformar arrays em elementos JSX:
  
    const itens = ["React", "Vue", "Angular"];
    const lista = itens.map((item) => <li key={item}>{item}</li>);
  
Sugestão de Extensão:
  Adicione números dinamicamente:
  
    const lista = itens.map((item, idx) => <li key={item}>{idx+1}. {item}</li>);
