Exercício 1: Minha Primeira Prop

  • Descrição: Crie um componente funcional chamado Saudacao que exiba uma mensagem de saudação. Esta mensagem deve ser passada para o componente através de uma prop chamada mensagem. Exiba a mensagem dentro de um <h2>.
  • Conceitos Teóricos:
    • Props (abreviação de "properties") são argumentos que você passa para componentes React. Elas permitem que você envie dados de um componente pai para um componente filho, tornando os componentes reutilizáveis e dinâmicos.
    • Componentes funcionais recebem as props como um único objeto JavaScript no seu primeiro argumento.

Exemplo de Código:

// App.js
import React from 'react';

function Saudacao(props) {
  return <h2>{props.mensagem}</h2>;
}

function App() {
  return (
    <div>
      <Saudacao mensagem="Olá, mundo do React!" />
    </div>
  );
}

export default App;



Dicas:
Lembre-se que as props são acessadas como propriedades de um objeto. Por exemplo, se você tem uma prop chamada nome, você a acessa como props.nome.
Sugestão de Extensão:
Crie uma nova instância do componente Saudacao passando uma mensagem diferente.
