Exercício 3: Renderização Condicional com Props

  • Descrição: Crie um componente StatusOnline que receba uma prop booleana isOnline. Se isOnline for true, o componente deve exibir "Usuário Online". Caso contrário, deve exibir "Usuário Offline". Use uma renderização condicional simples.
  • Conceitos Teóricos:
    • Você pode usar expressões JavaScript dentro do JSX, incluindo operadores ternários (condicao ? valorSeVerdadeiro : valorSeFalso) para renderização condicional baseada em props.

Exemplo de Código:

// App.js
import React from 'react';

function StatusOnline(props) {
  return (
    <div>
      {props.isOnline ? <p>Usuário Online</p> : <p>Usuário Offline</p>}
    </div>
  );
}

function App() {
  return (
    <div>
      <StatusOnline isOnline={true} />
      <StatusOnline isOnline={false} />
    </div>
  );
}

export default App;



Dicas:
Lembre-se que booleanos também são passados entre chaves {}.
Sugestão de Extensão:
Adicione um estilo diferente (cor, por exemplo) para a mensagem "Online" e "Offline".
