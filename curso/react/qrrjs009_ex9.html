Exercício 9: Compartilhando Função com useContext

  • Descrição: Baseado no Exercício 8, modifique o ComponenteFilho para que ele tenha um botão. Ao clicar neste botão, o tema deve ser alternado. Para isso, você precisará passar a função de alternar o tema do App para o ComponenteFilho usando o useContext.
  • Conceitos Teóricos:
    • Compartilhando Funções via Contexto: Assim como dados, funções também podem ser passadas através do Contexto, permitindo que componentes descendentes acionem lógicas definidas em componentes ancestrais.

Dicas:
O value do TemaContext.Provider pode ser um objeto contendo tanto o tema quanto a função toggleTema.
