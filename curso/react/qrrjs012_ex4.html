Exercício 4: Estilos Dinâmicos com Props

  • Descrição: Crie um componente Alerta que recebe uma prop tipo ('sucesso', 'erro', 'aviso'). O componente deve exibir uma mensagem com um estilo diferente (cor de fundo e cor do texto) com base no tipo recebido. Use estilos inline.
  • Conceitos Teóricos:
    • Você pode usar props para passar dados para seus componentes. Combinando isso com estilos inline, você pode criar estilos dinâmicos baseados nas props.

Exemplo de Código:

import React from 'react';

function Alerta({ tipo, mensagem }) {
const estilos = {
padding: '10px',
borderRadius: '5px',
marginBottom: '10px',
color: 'white',
backgroundColor: '#ccc' // Cor padrão
};

if (tipo === 'sucesso') {
estilos.backgroundColor = '#28a745'; // Verde
} else if (tipo === 'erro') {
estilos.backgroundColor = '#dc3545'; // Vermelho
} else if (tipo === 'aviso') {
estilos.backgroundColor = '#ffc107'; // Amarelo
estilos.color = '#333'; // Texto escuro para contraste
}

return (


{mensagem}


);
}

export default Alerta;

Como usar:






Dica:
Use um if/else if ou um switch para aplicar diferentes estilos com base na prop tipo.
Sugestão de Extensão:
Adicione uma prop icone para exibir um ícone diferente dependendo do tipo do alerta.
