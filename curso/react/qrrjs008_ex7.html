Exercício 7: Troca de Tema (Dark/Light Mode)

  • Descrição: Crie um componente com um botão. Ao clicar no botão, a cor de fundo do componente deve alternar entre claro (por exemplo, branco) e escuro (por exemplo, preto). O texto do componente deve mudar para uma cor que contraste com o fundo.
  • Conteúdo Teórico:
    • Estilos Dinâmicos: Você pode aplicar estilos CSS dinamicamente usando objetos JavaScript para a prop style.
    • Múltiplos Estados: É comum ter múltiplos estados em um componente, cada um controlando uma parte diferente da UI.

Dica:
Use um estado booleano para controlar qual tema está ativo e condicione os estilos com base nesse estado.
Sugestão de Extensão:
Adicione mais opções de tema (por exemplo, azul, verde) e permita que o usuário escolha entre elas usando botões ou um select.
