Exercício 2: Mudando o Texto de um Parágrafo

  • Descrição: Crie um componente que contenha um parágrafo com o texto inicial "Olá!". Adicione um botão que, ao ser clicado, mude o texto do parágrafo para "Texto Alterado!".
  • Conteúdo Teórico:
    • State Hook (useState): O useState é uma função especial do React (um Hook) que nos permite adicionar o "estado" a componentes de função. O estado é basicamente uma variável que o React "monitora". Quando o estado muda, o componente é renderizado novamente.
    • Como usar useState: Ele retorna um array com duas coisas: o valor atual do estado e uma função para atualizar esse estado. Ex: const [texto, setTexto] = useState('Olá!');
    • Atualizando o Estado: Para mudar o valor do estado, você sempre usa a função de atualização que o useState te deu (setTexto no exemplo acima). Nunca modifique o estado diretamente.

Exemplo de Código:

import React, { useState } from 'react';

function MeuComponente() {
  const [mensagem, setMensagem] = useState('Olá!');

  function handleClick() {
    setMensagem('Texto Alterado!');
  }

  return (
    

      
{mensagem}

      
    

  );
}



Dica:
Não se esqueça de importar useState do React!
