Exercício 8: Props com Styled Components

  • Descrição: Refatore o componente Alerta do Exercício 4 para usar Styled Components. O componente deve mudar sua cor de fundo com base na prop tipo ('sucesso', 'erro', 'aviso').
  • Conceitos Teóricos:
    • Styled Components permite acessar props diretamente dentro dos template literals usando uma função que recebe as props como argumento.

Exemplo de Código:

import React from 'react';
import styled from 'styled-components';

const AlertaContainer = styled.div`
padding: 10px;
border-radius: 5px;
margin-bottom: 10px;
color: white;
background-color: ${(props) => {
  if (props.tipo === 'sucesso') return '#28a745';
  if (props.tipo === 'erro') return '#dc3545';
  if (props.tipo === 'aviso') return '#ffc107';
  return '#ccc'; // Cor padrão
}};
color: ${(props) => (props.tipo === 'aviso' ? '#333' : 'white')};
`;

function Alerta({ tipo, mensagem }) {
return (

{mensagem}

);
}

export default Alerta;

Como usar:






Dica:
Dentro do template literal de um Styled Component, você pode usar uma arrow function que recebe as props do componente.
Sugestão de Extensão:
Crie um componente Botao com Styled Components que mude sua cor de fundo baseado em uma prop principal (true/false).
