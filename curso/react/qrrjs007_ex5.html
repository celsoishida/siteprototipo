Exercício 5: Lista de Itens Dinâmica

  • Descrição: Crie um componente ListaDeTarefas que exiba uma lista de tarefas. Inicialmente, a lista deve estar vazia. Adicione um campo de input e um botão "Adicionar". Ao digitar uma tarefa no input e clicar no botão, a tarefa deve ser adicionada à lista.
  • Conteúdo Teórico:
    • State com Arrays: O state pode conter arrays. Para adicionar ou remover itens de um array no state, você NUNCA deve modificar o array diretamente. Em vez disso, crie um novo array com as alterações e use a função de atualização do useState para substituí-lo. Isso garante que o React detecte a mudança e re-renderize o componente.
