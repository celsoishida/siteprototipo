Exercício 10: Componente Reutilizável de Busca de Dados

  • Descrição: Crie um componente genérico DataLoader que recebe uma url como prop e renderiza um componente filho (usando children ou uma render prop) com os dados obtidos. Este DataLoader deve lidar com os estados de carregamento e erro, passando os dados ou a mensagem de erro para o componente filho.
  • Conteúdo Teórico:
    • Componentes reutilizáveis são a base do React. Você pode criar componentes que encapsulam lógica comum, como a busca de dados.
    • Children Prop: Você pode passar um componente ou elementos JSX como children para um componente, e o componente pai os renderizará.
    • Render Prop: É uma técnica onde você passa uma função como prop para um componente, e essa função é responsável por renderizar parte da UI, recebendo dados do componente pai.

Dica:
Pense em como passar os dados (data, isLoading, error) para o componente filho de forma flexível. A render prop ou a children prop são ótimas para isso.
Sugestão de Extensão:
Adicione um prop ao DataLoader para que o usuário possa definir uma função de mapeamento de dados antes de serem passados para o componente filho.
