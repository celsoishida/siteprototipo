Exercício 2: Múltiplas Classes CSS Tradicionais

  • Descrição: Crie um componente Card que exiba um título e um parágrafo. Estilize-o usando duas classes CSS tradicionais: uma para o contêiner do card (.card-container) e outra para o título (.card-titulo), ambas no App.css.
  • Conceitos Teóricos:
    • Você pode aplicar múltiplas classes CSS a um elemento separando-as por espaços na propriedade className.

Exemplo de Código:
No App.css:

.card-container {
border: 1px solid #ccc;
padding: 15px;
margin: 10px;
border-radius: 8px;
box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.1);
}

.card-titulo {
color: #333;
font-size: 20px;
margin-bottom: 10px;
}

import React from 'react';
