Exercício 1: Buscando um Recurso Simples com fetch

  • Descrição: Crie um componente React que, ao ser montado, faça uma requisição HTTP GET para a API JSONPlaceholder e exiba o Id e a Descricao.
    Endpoint: http://200.17.199.251:5002/librasapi/cursosufpr
  • Conteúdo Teórico:
    • fetch é uma API web nativa do JavaScript para fazer requisições HTTP. Ele retorna uma Promise que é resolvida com a resposta da requisição.
    • O Hook useEffect é usado para executar efeitos colaterais em componentes funcionais. Quando o segundo argumento (array de dependências) é um array vazio ([]), o efeito é executado apenas uma vez, após a montagem inicial do componente, simulando o componentDidMount de classes.
    • O Hook useState permite adicionar estado a componentes funcionais.

Dica:
Lembre-se de converter a resposta para JSON usando response.json().
Sugestão de Extensão:
Exiba também o ID do todo.
