Exercício 10: Combinando Estilos (CSS Modules e Styled Components)

  • Descrição: Crie um componente chamado SecaoProduto. A seção principal (div externa) deve ser estilizada usando Styled Components, com uma borda e um padding. Dentro dela, o título do produto (h2) deve ser estilizado usando CSS Modules para definir a cor e o tamanho da fonte.
  • Conceitos Teóricos:
    • Em uma aplicação React, é comum que diferentes partes do seu código usem diferentes abordagens de estilização. Você pode, por exemplo, usar Styled Components para layout e temas globais, e CSS Modules para componentes mais específicos ou que precisam de isolamento de escopo.
    • A chave é saber como importar e aplicar os estilos de cada método corretamente.

/* TituloProduto.module.css */

color: #8A2BE2; /* Azul violeta */
font-size: 28px;
text-align: center;
margin-bottom: 15px;
}
