Exercício 9: Herança de Estilos com Styled Components

  • Descrição: Crie um componente BotaoPadrao que servirá como base. Em seguida, crie outros dois componentes, BotaoConfirmar e BotaoCancelar, que herdarão os estilos de BotaoPadrao, mas terão suas próprias cores de fundo e de texto.
  • Conceitos Teóricos:
    • Herança de Estilos: Com Styled Components, você pode criar um componente estilizado e depois reutilizar seus estilos como base para outros componentes. Isso promove a reusabilidade de código e facilita a manutenção, pois você define estilos comuns apenas uma vez.
    • A sintaxe para herdar estilos é styled(ComponenteBase).

Exemplo de Código:
Primeiro, garanta que styled-components esteja instalado:
npm install styled-components ou yarn add styled-components

No seu arquivo de componente (por exemplo, BotoesHerdados.jsx):

import React from 'react';
import styled from 'styled-components';

// 1. Componente Base com estilos comuns
const BotaoPadrao = styled.button`padding: 10px 20px; border-radius: 5px; border: none; cursor: pointer; font-weight: bold; margin: 5px; /* Adicionado para melhor visualização */`;

// 2. Botão de Confirmação que herda de BotaoPadrao
const BotaoConfirmar = styled(BotaoPadrao)`
background-color: #28a745; /* Verde */
color: white;

&:hover {
background-color: #218838; /* Verde mais escuro no hover */
}
`;

// 3. Botão de Cancelamento que herda de BotaoPadrao
const BotaoCancelar = styled(BotaoPadrao)`
background-color: #dc3545; /* Vermelho */
color: white;

&:hover {
background-color: #c82333; /* Vermelho mais escuro no hover */
}
`;

function BotoesHerdados() {
return (


Confirmar Ação
Cancelar Operação


);
}

export default BotoesHerdados;



Dica:
Pense no BotaoPadrao como um "molde" ou "modelo" para os outros botões. Ele define o que é comum entre eles, e os botões derivados apenas adicionam ou modificam o que é específico.
Sugestão de Extensão:
Crie um LinkEstilizado que herda os estilos de um BotaoPadrao e adicione estilos para remover o sublinhado e mudar a cor do texto.
