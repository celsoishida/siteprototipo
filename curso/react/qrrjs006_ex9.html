Exercício 9: Reutilização de Componentes com Props

  • Descrição: Crie um componente ItemDeLista que receba props titulo e descricao. Crie um componente ListaDeTarefas que utilize múltiplas instâncias de ItemDeLista para exibir uma lista de tarefas, cada uma com seu título e descrição únicos.
  • Conceitos Teóricos:
    • O poder das props reside na capacidade de criar componentes reutilizáveis. Você define a estrutura uma vez e, em seguida, passa dados diferentes para cada instância, adaptando seu comportamento e conteúdo.

Exemplo de Código:

// App.js
import React from 'react';

function ItemDeLista({ titulo, descricao }) {
  return (
    <div style={{ border: '1px dotted #eee', padding: '10px', margin: '5px' }}>
      <h4>{titulo}</h4>
      <p>{descricao}</p>
    </div>
  );
}

function ListaDeTarefas() {
  return (
    <div>
      <h3>Minhas Tarefas:</h3>
      <ItemDeLista titulo="Estudar React" descricao="Revisar conceitos de Props e State." />
      <ItemDeLista titulo="Fazer Exercícios" descricao="Completar os exercícios do curso." />
      <ItemDeLista titulo="Planejar Projeto" descricao="Pensar em ideias para um projeto pessoal." />
    </div>
  );
}

function App() {
  return (
    <div>
      <ListaDeTarefas />
    </div>
  );
}

export default App;



Dicas:
Pense em como você pode quebrar sua interface em componentes menores e reutilizáveis, cada um com sua responsabilidade.
Sugestão de Extensão:
Crie um array de objetos em ListaDeTarefas, onde cada objeto representa uma tarefa com titulo e descricao. Use map para renderizar os ItemDeLista.
