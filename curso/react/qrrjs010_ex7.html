Exercício 7: Página "Não Encontrado" (404)

  • Descrição: Crie uma rota coringa (`*`) no seu `App.js` que renderize um componente `NotFound.js` quando o usuário tentar acessar uma URL que não corresponde a nenhuma rota definida.
  • Conceitos Teóricos:
    • Rota Coringa (`*`): Uma rota com o path="*" (asterisco) atua como um "catch-all", capturando qualquer URL que não foi correspondida por nenhuma das rotas anteriores. É ideal para páginas de erro 404.

Exemplo de Código:

// src/components/NotFound.js
import React from 'react';
import { Link } from 'react-router-dom';

function NotFound() {
  return (
    <div>
      <h2>404 - Página Não Encontrada</h2>
      <p>Parece que você se perdeu. <Link to="/">Voltar para a Home</Link>.</p>
    </div>
  );
}

export default NotFound;

// src/App.js (atualizado)
import React from 'react';
import { Routes, Route, Link } from 'react-router-dom';
import Home from './components/Home';
import About from './components/About';
import Dashboard from './components/Dashboard';
import Profile from './components/Profile';
import Settings from './components/Settings';
import Products from './components/Products';
import ProductDetail from './components/ProductDetail';
import NotFound from './components/NotFound'; // Importe NotFound

function App() {
  return (
    <div>
      <h1>Meu App de Rotas</h1>
      <nav>
        <ul>
          <li><Link to="/">Home</Link></li>
          <li><Link to="/about">Sobre</Link></li>
          <li><Link to="/dashboard">Dashboard</Link></li>
          <li><Link to="/products">Produtos</Link></li>
        </ul>
      </nav>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
        <Route path="/dashboard" element={<Dashboard />}>
          <Route path="profile" element={<Profile />} />
          <Route path="settings" element={<Settings />} />
        </Route>
        <Route path="/products" element={<Products />} />
        <Route path="/products/:id" element={<ProductDetail />} />
        <Route path="*" element={<NotFound />} /> {/* Rota coringa no final */}
      </Routes>
    </div>
  );
}

export default App;



Dicas:
- A rota coringa (`*`) deve ser sempre a última rota definida dentro de `Routes`, pois as rotas são avaliadas em ordem.

Sugestão de Extensão:
- Adicione um contador regressivo na página `NotFound` que redirecione automaticamente para a página inicial após alguns segundos.
