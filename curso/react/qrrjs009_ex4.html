Exercício 4: To-Do List Simples

  • Descrição: Crie um componente de lista de tarefas (To-Do List) muito simples. Ele deve ter um campo de input para adicionar novas tarefas e um botão. Ao clicar no botão, a tarefa digitada deve ser adicionada a uma lista exibida abaixo.
  • Conceitos Teóricos:
    • useState com Arrays: Você pode usar useState para gerenciar listas de dados (arrays). Para adicionar um novo item a um array no estado, você precisa criar um novo array com o item adicionado e então usar a função de atualização do estado.

Dicas:
Para adicionar um item a um array no estado, utilize o operador spread (...) para criar uma cópia do array existente e adicione o novo item.
Ex: setTarefas([...tarefas, novaTarefa]);
Sugestão de Extensão:
Adicione um botão para remover tarefas da lista.
