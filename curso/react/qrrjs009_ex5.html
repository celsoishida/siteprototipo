Exercício 5: Entendendo useEffect - Montagem

  • Descrição: Crie um componente que, ao ser renderizado pela primeira vez (montagem), exiba uma mensagem no console dizendo "Componente montado!".
  • Conceitos Teóricos:
    • useEffect: É um Hook que permite executar "efeitos colaterais" em componentes funcionais. Efeitos colaterais são operações que não afetam diretamente a renderização do componente, como chamadas de API, manipulação do DOM ou configurações de timers.
    • Array de Dependências Vazio ([]): Quando o useEffect possui um array de dependências vazio, a função do efeito será executada apenas uma vez, após a primeira renderização do componente (equivalente ao componentDidMount em classes).

Exemplo de Código:

import React, { useEffect } from 'react';

function ComponenteMontagem() {
useEffect(() => {
console.log('Componente montado!');
}, []); // Array de dependências vazio: executa apenas na montagem

Abra o console para ver a mensagem de montagem.

}
