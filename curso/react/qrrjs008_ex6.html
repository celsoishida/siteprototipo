Exercício 6: Lista de Itens Adicionáveis

  • Descrição: Crie um componente que exiba uma lista de itens (por exemplo, <ul>). Adicione um campo de input e um botão "Adicionar". Ao digitar um texto no input e clicar no botão, o texto deve ser adicionado como um novo item na lista.
  • Conteúdo Teórico:
    • Arrays no Estado: Você pode armazenar arrays no estado do React. Para adicionar um novo item a um array no estado, você precisa criar um novo array que inclua o item antigo e o novo. Não use push() diretamente no array do estado, pois isso mutaria o estado.
    • Função de Atualização com Callback: Quando o novo estado depende do estado anterior (como adicionar um item a uma lista), é uma boa prática usar a forma de função para a função de atualização de estado: setLista(prevLista => [...prevLista, novoItem]). Isso garante que você esteja sempre trabalhando com o valor mais recente do estado.
    • Map para Renderizar Listas: Para renderizar uma lista de itens de um array, usamos o método map do JavaScript. Lembre-se de fornecer uma key única para cada item renderizado em uma lista.
