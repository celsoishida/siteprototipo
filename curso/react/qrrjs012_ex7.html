Exercício 7: Introdução ao Styled Components

  • Descrição: Refatore o componente BotaoSimples novamente, mas desta vez usando Styled Components. Crie um botão azul com texto branco.
  • Conceitos Teóricos:
    • Styled Components: É uma biblioteca que permite escrever CSS dentro do seu código JavaScript, criando componentes React com estilos já acoplados. Isso oferece estilização com escopo e reusabilidade.
    • Instalação: Você precisará instalar styled-components (npm install styled-components ou yarn add styled-components).
    • Sintaxe: Use a sintaxe de tagged template literals (strings com acento grave ` ` ) após styled. para definir os estilos.

Exemplo de Código:
Primeiro, instale styled-components:
npm install styled-components ou yarn add styled-components

No seu componente BotaoSimples.jsx:

import React from 'react';
import styled from 'styled-components'; // Importe styled

const BotaoEstilizado = styled.button`
background-color: #007bff;
color: white;
padding: 10px 20px;
border-radius: 5px;
border: none;
cursor: pointer;

&:hover { /* Estilo de hover diretamente no CSS */
background-color: #0056b3;
}
`;
