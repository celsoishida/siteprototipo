Exercício 6: Props com Valores Padrão

  • Descrição: Crie um componente Botao que receba uma prop texto. Se a prop texto não for fornecida, o botão deve exibir "Clique aqui".
  • Conceitos Teóricos:
    • Você pode definir valores padrão para props de duas maneiras principais:
    • Dentro da desestruturação: Usando o operador de atribuição de valor padrão (=) diretamente na desestruturação das props. Ex: function Botao({ texto = "Valor Padrão" }) { ... }.
    • defaultProps: Uma propriedade estática no componente que define valores padrão. Embora ainda funcione, a desestruturação com valores padrão é mais moderna e preferida para componentes funcionais.

Exemplo de Código:

// App.js
import React from 'react';

function Botao({ texto = "Clique aqui" }) {
  return <button>{texto}</button>;
}

function App() {
  return (
    <div>
      <Botao texto="Enviar" />
      <Botao /> {/* Não passa a prop texto */}
    </div>
  );
}

export default App;



Dicas:
Valores padrão são úteis para tornar seus componentes mais robustos e flexíveis.
Sugestão de Extensão:
Adicione uma prop cor ao botão com um valor padrão, por exemplo, "azul".
