Exercício 9: Exibindo Feedback do Usuário

  • Descrição: Crie um componente com um botão. Ao clicar no botão, exiba uma mensagem de "Sucesso!" que desapareça automaticamente após 3 segundos.
  • Conteúdo Teórico:
    • setTimeout com useState: Você pode usar setTimeout para agendar uma função que será executada após um determinado atraso. Isso é útil para exibir mensagens temporárias. Lembre-se de que o setTimeout é um efeito colateral e, em componentes mais complexos, o useEffect seria a escolha ideal, mas para este exercício básico, o setTimeout direto em um manipulador de eventos funciona bem.

Dica:
Use um estado booleano para controlar a visibilidade da mensagem de sucesso. Quando o botão for clicado, defina esse estado como true e use setTimeout para defini-lo de volta para false após o tempo desejado.
Sugestão de Extensão:
Adicione uma mensagem de "Erro!" que apareça em vermelho caso haja algum problema (simule um erro com um Math.random()).
