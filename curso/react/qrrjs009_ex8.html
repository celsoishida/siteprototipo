Exercício 8: Introdução ao useContext

  • Descrição: Crie dois componentes: App e ComponenteFilho. O componente App deve ter um estado chamado tema (por exemplo, 'claro' ou 'escuro'). Passe este estado para o ComponenteFilho usando useContext, de modo que o ComponenteFilho possa exibir o tema atual.
  • Conceitos Teóricos:
    • useContext: É um Hook que permite a componentes funcionais subscreverem a um Contexto React. O Contexto é uma forma de passar dados através da árvore de componentes sem ter que passar props manualmente em cada nível.
    • createContext: Usado para criar um objeto Contexto.
    • Provider: O componente Provider do Contexto permite que componentes descendentes assinem as mudanças de um Contexto.
