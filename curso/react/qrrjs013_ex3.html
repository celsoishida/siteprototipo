Exercício 3: Lidando com Estados de Carregamento

  • Descrição: Modifique o componente do Exercício 2 para exibir uma mensagem de "Carregando..." enquanto os posts estão sendo buscados e uma mensagem de erro caso a requisição falhe.
    Endpoint: https://jsonplaceholder.typicode.com/posts?_limit=5
  • Conteúdo Teórico:
    • É comum usar variáveis de estado para gerenciar o ciclo de vida de uma requisição:
      • isLoading (booleano): Indica se a requisição está em andamento.
      • isError (booleano) ou error (objeto): Indica se houve um erro na requisição e armazena os detalhes do erro.
    • Você pode usar renderização condicional para mostrar diferentes partes da UI com base nesses estados.


Dica:
Inicialize isLoading como true e defina-o como false após o sucesso ou falha da requisição.
Sugestão de Extensão:
Exiba uma mensagem de erro mais detalhada, por exemplo, o message do objeto error.
