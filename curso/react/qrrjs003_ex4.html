Exercício 4: Componentes Funcionais

  • Descrição: Crie um componente funcional que exiba um título com um subtítulo.
  • Conteúdo Teórico: Um componente funcional é uma função que retorna JSX.

Exemplo de Código:

function Titulo() {
  return <h1>Olá, React!</h1>;
}

Dica: Componentes funcionais começam com letra maiúscula.
Sugestão de Extensão: Crie um segundo componente para um parágrafo. E mude de ordem as mensagens.
