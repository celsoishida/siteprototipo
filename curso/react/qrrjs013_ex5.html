Exercício 5: Buscando Dados com Base em Interação do Usuário

  • Descrição: Crie um componente que exiba um input de texto e um botão. Quando o usuário digitar um número (ID do usuário) no input e clicar no botão, o componente deve buscar os "posts" desse usuário específico na API JSONPlaceholder e exibi-los.
    Endpoint: https://jsonplaceholder.typicode.com/posts?userId=[ID_DO_USUARIO]
  • Conteúdo Teórico:
    • Para capturar o valor de um input, use o atributo value e o evento onChange junto com o useState.
    • Para disparar uma requisição em resposta a um evento (como um clique de botão), você pode criar uma função assíncrona e chamá-la no manipulador de eventos.


Dica:
A requisição deve ser feita dentro de uma função disparada pelo botão, não no useEffect com array de dependências vazio, pois ela depende da ação do usuário.
Sugestão de Extensão:
Adicione validação para garantir que o input seja um número válido antes de fazer a requisição.
