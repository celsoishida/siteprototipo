Exercício 6: Consumindo uma API com Parâmetros Dinâmicos

  • Descrição: Crie um componente que exiba um dropdown (select) com opções de IDs de usuários (por exemplo, 1, 2, 3, 4, 5). Quando o usuário selecionar um ID, o componente deve buscar os "albums" desse usuário específico na API JSONPlaceholder e listá-los.
    Endpoint: https://jsonplaceholder.typicode.com/users/[ID_DO_USUARIO]/albums
  • Conteúdo Teórico:
    • O Hook useState pode ser usado para armazenar o valor selecionado de um select.
    • O useEffect pode ser usado para disparar a requisição sempre que o ID do usuário selecionado mudar. Adicione o estado do ID do usuário como dependência no array de dependências do useEffect.


Dica:
Você pode usar o event.target.value no onChange do select para obter o valor selecionado.
Sugestão de Extensão:
Além do nome do álbum, exiba também o número de fotos em cada álbum (se houver essa informação disponível na API, caso contrário, invente um número aleatório).
