Exercício 4: Input Controlado

  • Descrição: Crie um componente FormularioSimples que contenha um campo de input de texto e um parágrafo. À medida que o usuário digita no input, o texto digitado deve ser exibido em tempo real no parágrafo.
  • Conteúdo Teórico:
    • Inputs Controlados: No React, um "input controlado" é um elemento de formulário cujo valor é controlado pelo state do React. Isso significa que o React é a "fonte da verdade" para o valor do input, e você o atualiza através do state.
    • onChange: É um evento que é disparado sempre que o valor de um input é alterado. Você pode usar este evento para atualizar o state com o novo valor do input.

Exemplo de Código:

// App.js
import React, { useState } from 'react';

function FormularioSimples() {
  const [textoDigitado, setTextoDigitado] = useState('');

  const handleChange = (event) => {

Dica:
O event.target.value dentro do onChange é crucial para pegar o valor atual do input.
