Exercício 7: children Prop

  • Descrição: Crie um componente funcional chamado Quadro que receba e renderize sua children prop. No componente principal, use Quadro e, dentro dele, coloque um <h2> com "Título do Quadro" e um <p> com "Este é o conteúdo do quadro.".
  • Conteúdo Teórico:
    • A prop especial children permite que você passe elementos e componentes diretamente entre as tags de abertura e fechamento de um componente.
    • Qualquer coisa que você renderize entre a tag de abertura e fechamento de um componente React será passado para esse componente como sua prop children.
    • É útil para componentes que atuam como "containers" ou "layouts".

Dica:
Dentro do componente Quadro, você pode simplesmente renderizar {props.children} para exibir o conteúdo passado.
Sugestão de Extensão:
Adicione um estilo CSS básico ao componente Quadro (ex: uma borda) para visualmente delimitar o conteúdo.
