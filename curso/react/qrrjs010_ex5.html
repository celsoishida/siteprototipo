Exercício 5: Parâmetros de Rota (`URL Parameters`)

  • Descrição: Crie uma página `ProductDetail.js` que exiba informações de um produto com base em um ID passado na URL (ex: `/products/123`). Adicione um link para essa página a partir de um componente `Products.js`.
  • Conceitos Teóricos:
    • Parâmetros de Rota: Permitem capturar valores dinâmicos na URL. Você define um parâmetro prefixando-o com `:` (ex: `:id`).
    • useParams: Hook do `react-router-dom` para acessar os parâmetros de rota.

Exemplo de Código:

// src/components/Products.js
import React from 'react';
import { Link } from 'react-router-dom';

function Products() {
  const products = [
    { id: '1', name: 'Notebook' },
    { id: '2', name: 'Mouse' },
    { id: '3', name: 'Teclado' },
  ];

  return (
    <div>
      <h2>Nossos Produtos</h2>
      <ul>
        {products.map(product => (
          <li key={product.id}>
            <Link to={`/products/${product.id}`}>{product.name}</Link>
          </li>
        ))}
      </ul>
    </div>
  );
}

export default Products;

// src/components/ProductDetail.js
import React from 'react';
import { useParams } from 'react-router-dom';

function ProductDetail() {
  const { id } = useParams(); // Captura o parâmetro 'id' da URL

  // Aqui você faria uma requisição para buscar os detalhes do produto com base no ID
  const product = { id: id, name: `Produto ${id}`, description: `Detalhes do Produto ${id}.` };

  return (
    <div>
      <h2>Detalhes do Produto: {product.name}</h2>
      <p>ID: {product.id}</p>
      <p>{product.description}</p>
    </div>
  );
}

export default ProductDetail;

// src/App.js (atualizado)
import React from 'react';
import { Routes, Route, Link } from 'react-router-dom';
import Home from './components/Home';
import About from './components/About';
import Dashboard from './components/Dashboard';
import Profile from './components/Profile';
import Settings from './components/Settings';
import Products from './components/Products'; // Importe Products
import ProductDetail from './components/ProductDetail'; // Importe ProductDetail

function App() {
  return (
    <div>
      <h1>Meu App de Rotas</h1>
      <nav>
        <ul>
          <li><Link to="/">Home</Link></li>
          <li><Link to="/about">Sobre</Link></li>
          <li><Link to="/dashboard">Dashboard</Link></li>
          <li><Link to="/products">Produtos</Link></li> {/* Adicione o link para Produtos */}
        </ul>
      </nav>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
        <Route path="/dashboard" element={<Dashboard />}>
          <Route path="profile" element={<Profile />} />
          <Route path="settings" element={<Settings />} />
        </Route>
        <Route path="/products" element={<Products />} /> {/* Rota para lista de produtos */}
        <Route path="/products/:id" element={<ProductDetail />} /> {/* Rota com parâmetro */}
      </Routes>
    </div>
  );
}

export default App;



Dicas:
- O nome do parâmetro na rota (`:id`) deve ser o mesmo usado no `useParams({ id })`.

Sugestão de Extensão:
- Crie um array de objetos `products` no `Products.js` e passe-o como `prop` para `ProductDetail` para simular a busca de dados.
