Exercício 10: Toggle de Tema com State

  • Descrição: Crie um componente SeletorDeTema com um botão que, ao ser clicado, alterne o tema da sua aplicação (por exemplo, entre "claro" e "escuro"). Você pode fazer isso mudando uma classe CSS no elemento body ou em algum elemento pai do seu componente.
  • Conteúdo Teórico:
    • Interação com o DOM (via State): Embora o React priorize a manipulação do DOM através do state, em alguns casos, pode ser necessário interagir diretamente com o DOM (por exemplo, para adicionar classes ao body). No entanto, a decisão de qual classe aplicar ainda deve ser baseada no state do componente.

Exemplo de Código:

// App.js
import React, { useState, useEffect } from 'react';

function SeletorDeTema() {
  const [temaEscuro, setTemaEscuro] = useState(false);

  useEffect(() => {
    document.body.className = temaEscuro ? 'dark-theme' : 'light-theme';
  }, [temaEscuro]);

  return (
    
  );
}

export default SeletorDeTema;



Dica:
Use um state booleano para controlar o tema atual. Você pode adicionar ou remover classes CSS dinamicamente com base nesse state.
Sugestão de Extensão:
Crie estilos CSS para os temas "claro" e "escuro" e veja a sua aplicação mudar de visual!
