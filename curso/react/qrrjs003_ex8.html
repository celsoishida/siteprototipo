Exercício 8. Estilização com CSS Inline e Externo

  • Descrição: Aplique estilos ao JSX usando CSS inline.
  • Conteúdo Teórico: Podemos estilizar JSX com objetos JavaScript ou classes externas.
  • Exemplo de Código: 
    
const estilo = { color: "blue", fontSize: "20px" };
const elemento = <h1 style={estilo}>Texto estilizado</h1>;
  • Dica: Use style={{ propriedade: valor }} para estilos inline.
  • Sugestão de Extensão: Crie uma classe CSS e use className.

Dicas:
  Para CSS inline, use um objeto JS:
  
    const estilo = { color: "blue", fontSize: "20px" };
    const elemento = <h1 style={estilo}>Texto estilizado</h1>;
  
Sugestão de Extensão:
  Crie uma classe CSS e use className:
  
    .meu-titulo { color: red; font-size: 24px; }
    const elemento = <h1 className="meu-titulo">Texto estilizado</h1>;
