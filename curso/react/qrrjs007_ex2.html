Exercício 2: Botão de Contagem

  • Descrição: Crie um componente Contador que exiba um número e um botão. Cada vez que o botão for clicado, o número exibido deve ser incrementado em 1.
  • Conteúdo Teórico:
    • Atualização de State: Para alterar o valor de um state, você deve usar a função de atualização fornecida pelo useState. Chamar essa função irá re-renderizar o componente com o novo valor do state.

Exemplo de Código:

// App.js
import React, { useState } from 'react';

function Contador() {
  const [contagem, setContagem] = useState(0);

  return (
    

      
Contagem: {contagem}

      
    

  );
}

export default Contador;



Dica:
Use o onClick do botão para chamar a função que atualiza o state!
Sugestão de Extensão:
Adicione um segundo botão para decrementar a contagem.
