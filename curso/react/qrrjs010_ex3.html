Exercício 3: Navegação com Link

  • Descrição: Adicione links de navegação no seu App.js para as páginas Home e About usando o componente Link do react-router-dom.
  • Conceitos Teóricos:
    • Link: O componente preferido para navegação interna em uma aplicação React. Ele impede o recarregamento completo da página, proporcionando uma experiência de usuário mais fluida.
    • to: Propriedade do Link que especifica o caminho para o qual o link deve navegar.
