Exercício 7: Múltiplos States Independentes

  • Descrição: Crie um componente PerfilDeUsuario que exiba o nome do usuário e sua idade. Adicione dois inputs separados: um para o nome e outro para a idade. Ao digitar em cada input, o respectivo dado no perfil deve ser atualizado.
  • Conteúdo Teórico:
    • Múltiplos useState: Você pode usar múltiplos useState em um único componente para gerenciar diferentes partes do seu state de forma independente. Isso torna o código mais legível e organizado.

Exemplo de Código:

// App.js
import React, { useState } from 'react';

function PerfilDeUsuario() {
  const [nome, setNome] = useState('');
  const [idade, setIdade] = useState('');

Crie um useState separado para o nome e outro para a idade.
