Exercício 2: Consumindo Contexto com useContext para Dados Simples

  • Descrição: Utilizando o ThemeContext criado no exercício anterior, crie dois componentes irmãos (Header e Footer). Ambos os componentes devem consumir o ThemeContext para exibir o tema atual.
  • Conceitos Teóricos:
    • useContext Hook: É um hook do React que permite que um componente de função se inscreva em um contexto. Ele recebe o objeto de contexto como argumento e retorna o valor atual desse contexto para o componente.
    • Este hook simplifica o acesso ao contexto em componentes funcionais, tornando o código mais limpo do que as abordagens baseadas em classes.

Exemplo de Código:

// ThemeContext.js (o mesmo do exercício 1)
import React, { createContext } from 'react';

export const ThemeContext = createContext('light');

// Header.js
import React, { useContext } from 'react';
import { ThemeContext } from './ThemeContext';

function Header() {
  const theme = useContext(ThemeContext);
  return <header style={{ background: theme === 'dark' ? '#333' : '#eee', color: theme === 'dark' ? '#eee' : '#333' }}>Cabeçalho - Tema: {theme}</header>;
}

// Footer.js
import React, { useContext } from 'react';
import { ThemeContext } from './ThemeContext';

function Footer() {
  const theme = useContext(ThemeContext);
  return <footer style={{ background: theme === 'dark' ? '#333' : '#eee', color: theme === 'dark' ? '#eee' : '#333' }}>Rodapé - Tema: {theme}</footer>;
}

// App.js
import React from 'react';
import { ThemeContext } from './ThemeContext';
import Header from './Header';
import Footer from './Footer';

function App() {
  return (
    <ThemeContext.Provider value="dark">
      <Header />
      <main>Conteúdo principal</main>
      <Footer />
    </ThemeContext.Provider>
  );
}



Dica:
Verifique se o ThemeProvider está envolvendo todos os componentes que precisam acessar o contexto.

Sugestão de Extensão:
Adicione um botão no Header ou Footer que alterne o tema de 'light' para 'dark' e vice-versa. Para isso, o provedor precisará gerenciar um estado para o tema.
