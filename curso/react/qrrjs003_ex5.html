Exercício 5: Uso de Props

  • Descrição: Passe um nome como propriedade (prop) para um componente.
  • Conteúdo Teórico: Props permitem passar dados de um componente pai para um componente filho.

Exemplo de Código:

function Saudacao(props) {
  return <h1>Olá, {props.nome}!</h1>;
}

Dica: Passe valores para o componente assim: <Saudacao nome="React" />.
Sugestão de Extensão: Crie um prop idade e exiba no componente.
