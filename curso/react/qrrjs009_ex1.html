Exercício 1: Olá, useState!

  • Descrição: Crie um componente funcional no React que exiba um contador. O contador deve começar em 0 e ter um botão "Incrementar" que, ao ser clicado, aumente o valor do contador em 1.
  • Conceitos Teóricos:
    • useState: É um Hook que permite adicionar estado a componentes funcionais. Ele retorna um par de valores: o estado atual e uma função para atualizá-lo.
    • Estado: No React, o estado representa os dados que um componente pode gerenciar e que, quando alterados, podem causar uma nova renderização do componente.

Exemplo de Código:

import React, { useState } from 'react';

function Contador() {
// Inicializa o estado 'contador' com 0
const [contador, setContador] = useState(0);

const handleIncrementar = () => {
// Atualiza o estado 'contador'
setContador(contador + 1);
};

return (



Contador: {contador}




);
}

export default Contador;



Dicas:
Lembre-se que o useState retorna um array com dois elementos: o valor atual do estado e a função para atualizá-lo.
Sugestão de Extensão:
Adicione um botão para decrementar o contador.
