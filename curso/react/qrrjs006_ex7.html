Exercício 7: Children Prop

  • Descrição: Crie um componente Card que receba qualquer conteúdo aninhado (filhos) como uma prop especial chamada children. O Card deve envolver esse conteúdo em um <div> com uma borda simples.
  • Conceitos Teóricos:
    • A prop especial children permite que você passe elementos ou componentes para dentro de outros componentes, tornando-os como "envelopes" ou "contêineres".
    • O conteúdo entre as tags de abertura e fechamento de um componente React (por exemplo, <Card>Este é o conteúdo do card</Card>) é acessível via props.children.

Exemplo de Código:

// App.js
import React from 'react';

function Card({ children }) {
  const cardStyle = {
    border: '1px solid #ccc',
    padding: '15px',
    margin: '10px',
    borderRadius: '5px'
  };

  return (
    <div style={cardStyle}>
      {children}
    </div>
  );
}

function App() {
  return (
    <div>
      <Card>
        <h3>Título do Card</h3>
        <p>Este é o conteúdo principal do meu card.</p>
      </Card>

      <Card>
        <p>Um card mais simples com apenas um parágrafo.</p>
      </Card>
    </div>
  );
}

export default App;



Dicas:
A prop children é superpoderosa para criar layouts flexíveis e componentes reutilizáveis.
Sugestão de Extensão:
Adicione uma prop titulo ao Card e exiba-a acima do children.
