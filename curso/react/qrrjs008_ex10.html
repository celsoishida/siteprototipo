Exercício 10: Tabs Simples

  • Descrição: Crie um componente com três botões, cada um representando uma aba (por exemplo, "Aba 1", "Aba 2", "Aba 3"). Abaixo dos botões, deve haver uma área de conteúdo. Ao clicar em um botão, o conteúdo correspondente àquela aba deve ser exibido, e o conteúdo das outras abas deve ser escondido.
  • Conteúdo Teórico:
    • Estado para Abas Ativas: Use um estado para manter qual aba está "ativa" no momento. Pode ser um número, uma string com o nome da aba, etc.
    • Renderização Condicional Avançada: Combine a renderização condicional com o estado da aba ativa para exibir o conteúdo correto.

Dica:
Use o estado para guardar qual aba está selecionada e use operadores ternários ou && para renderizar condicionalmente o conteúdo de cada aba.
Sugestão de Extensão:
Adicione um estilo visual aos botões das abas para indicar qual aba está ativa (por exemplo, mudar a cor de fundo do botão da aba ativa).
