Exercício 3: Estilos Inline

  • Descrição: Crie um componente `TextoColorido` que exiba um parágrafo. Use estilos inline para definir a cor do texto como vermelho e o tamanho da fonte como 18px.
  • Conceitos Teóricos:
    • Estilos Inline: Em React, você pode aplicar estilos diretamente nos elementos usando a propriedade `style`. O valor deve ser um objeto JavaScript onde as chaves são as propriedades CSS em `camelCase` e os valores são as strings de estilo.

Exemplo de Código:


function TextoColorido() {
    const estiloDoTexto = {
        color: 'red',
        fontSize: '18px'
    };

    return (
        <p style={estiloDoTexto}>
            Este texto é vermelho e tem 18 pixels de tamanho!
        </p>
    );
}

ReactDOM.render(<TextoColorido />, 
    document.getElementById('root'));

Dica:

Propriedades CSS com hífen (ex: `font-size`) são convertidas para `camelCase` (ex: `fontSize`) em estilos inline.

Sugestão de Extensão:

Crie um estado para a cor do texto e um botão que, ao ser clicado, mude a cor do parágrafo.
