Exercício 8: Aninhamento de Componentes de Classe

  • Descrição: Crie um componente de classe Painel que contenha um componente de classe Titulo (com um <h2> "Bem-vindo ao Painel") e um componente de classe Conteudo (com um <p> "Aqui está seu conteúdo personalizado."). Renderize o Painel no seu componente principal.
  • Conteúdo Teórico:
    • Assim como os componentes funcionais, os componentes de classe também podem ser aninhados e compostos.
    • A estrutura de importação e uso é a mesma: importe os componentes e inclua-os no método render() do componente pai.

Dica:
Lembre-se de que cada componente de classe precisa estender React.Component e ter um método render().
Sugestão de Extensão:
Passe uma prop para o componente Conteudo para que ele exiba uma mensagem diferente.
