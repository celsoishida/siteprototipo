Exercício 1: Nosso Primeiro Botão Interativo

  • Descrição: Crie um componente React que exiba um botão. Ao clicar no botão, ele deve exibir um alerta simples no navegador com a mensagem "Botão clicado!".
  • Conteúdo Teórico:
    • Eventos em React: No React, lidamos com eventos de forma muito semelhante ao JavaScript tradicional, mas com algumas diferenças importantes. Em vez de onclick="minhaFuncao()", usamos onClick={minhaFuncao} (observe o camelCase e o uso de chaves para passar a função).
    • Funções de Manipulação de Eventos: Geralmente, definimos funções separadas para lidar com os eventos. Essas funções são passadas como props para os elementos JSX.
