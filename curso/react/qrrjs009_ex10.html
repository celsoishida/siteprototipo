Exercício 10: Aplicação de Lista de Compras com Hooks

  • Descrição: Crie uma aplicação de lista de compras simples. Ela deve permitir:
    1. Adicionar novos itens à lista (usando useState para o input e para a lista de itens).
    2. Marcar itens como comprados (opcional, mas um bom desafio com useState para cada item).
    3. Exibir uma mensagem no console cada vez que a lista de compras for atualizada (usando useEffect).
  • Conceitos Teóricos:
    • Combinação de Hooks: Este exercício integra useState para gerenciamento de múltiplos estados e useEffect para observação de mudanças.

Dicas:
Para marcar itens como comprados, você pode armazenar objetos no array de itens, onde cada objeto tem um nome e um comprado: boolean.
Sugestão de Extensão:
Adicione um botão para remover itens da lista.
Use localStorage para persistir a lista de compras, para que ela não se perca ao recarregar a página (combine com useEffect!).
