Exercício 2: Altere a Cor do Texto

  • Descrição: Crie um componente que exiba um parágrafo de texto. Abaixo do parágrafo, adicione um botão. Quando o botão for clicado, a cor do texto do parágrafo deve alternar entre azul e vermelho.
  • Conceitos Teóricos:
    • useState com booleanos: Você pode usar useState para gerenciar estados booleanos (verdadeiro/falso) para controlar a exibição ou o estilo de elementos.

Dicas:
Use um estado booleano para controlar qual cor está ativa e aplique estilos inline ou classes CSS condicionais.
Sugestão de Extensão:
Adicione mais opções de cores e botões para cada cor.
