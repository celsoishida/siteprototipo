Exercício 5: Introdução aos CSS Modules

  • Descrição: Refatore o componente BotaoSimples do Exercício 1 para usar CSS Modules. Crie um arquivo BotaoSimples.module.css e aplique os estilos definidos nele.
  • Conceitos Teóricos:
    • CSS Modules: Resolvem o problema de escopo global do CSS tradicional, "localizando" as classes CSS. Quando você importa um arquivo .module.css, as classes são exportadas como propriedades de um objeto JavaScript, gerando nomes de classe únicos automaticamente (ex: BotaoSimples_botao-simples__xyz123).
    • Importação: Você importa o arquivo CSS Module como um objeto (ex: import styles from './BotaoSimples.module.css';).

Exemplo de Código:
Crie um arquivo BotaoSimples.module.css:

/* BotaoSimples.module.css */
.botaoSimples {
background-color: #007bff;
color: white;
padding: 10px 20px;
border-radius: 5px;
border: none;
cursor: pointer;
}

No seu componente BotaoSimples.jsx:

import React from 'react';
import styles from './BotaoSimples.module.css'; // Importe o CSS Module

}
