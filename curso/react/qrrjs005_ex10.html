Exercício 10: Componentes Aninhados com Props Múltiplas

  • Descrição: Crie um componente funcional ListaDeItens que receba uma prop titulo e uma prop itens (um array de strings). O componente ListaDeItens deve renderizar um <h2> com o título e uma lista não ordenada <ul> onde cada item do array itens é um <li>. No seu componente principal, use ListaDeItens duas vezes, uma para uma lista de "Frutas" e outra para uma lista de "Vegetais", cada uma com seus respectivos arrays de itens.
  • Conteúdo Teórico:
    • Este exercício combina tudo o que você aprendeu: componentes funcionais, props (incluindo arrays) e composição.
    • É comum passar arrays de dados para componentes filhos, que então "mapeiam" esses arrays para renderizar listas de elementos.

Dica:
Para renderizar uma lista de itens a partir de um array, você pode usar o método map() do JavaScript dentro do JSX. Lembre-se de fornecer uma key única para cada item da lista ao usar map().
Sugestão de Extensão:
Faça com que cada item da lista seja um componente funcional ItemLista que receba o texto do item como uma prop.
