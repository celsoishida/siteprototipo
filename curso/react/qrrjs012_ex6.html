Exercício 6: Múltiplas Classes com CSS Modules

  • Descrição: Refatore o componente Card do Exercício 2 para usar CSS Modules. Crie um arquivo Card.module.css e aplique as classes para o contêiner e o título.
  • Conceitos Teóricos:
    • Para aplicar múltiplas classes de CSS Modules, você pode usar template literals ou concatenar as classes do objeto styles.

Exemplo de Código:
Crie um arquivo Card.module.css:

/* Card.module.css */
.cardContainer {
border: 1px solid #ccc;
padding: 15px;
margin: 10px;
border-radius: 8px;
box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.1);
}

.cardTitulo {
color: #333;
font-size: 20px;
margin-bottom: 10px;
}

.cardParagrafo {
color: #666;
}
