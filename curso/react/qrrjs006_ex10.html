Exercício 10: Props Aninhadas (Nested Props)

  • Descrição: Crie um componente DadosContato que receba uma prop contato, que será um objeto com as propriedades nome, email e telefone. O componente deve exibir essas informações.
  • Conceitos Teóricos:
    • Props podem ser qualquer tipo de dado JavaScript, incluindo objetos e arrays. Você pode passar objetos complexos como props e acessar suas propriedades no componente filho.

Exemplo de Código:

// App.js
import React from 'react';

function DadosContato({ contato }) {
  return (
    <div>
      <h4>Contato:</h4>
      <p>Nome: {contato.nome}</p>
      <p>Email: {contato.email}</p>
      <p>Telefone: {contato.telefone}</p>
    </div>
  );
}

function App() {
  const meuContato = {
    nome: "João Silva",
    email: "joao.silva@example.com",
    telefone: "11987654321"
  };

  return (
    <div>
      <DadosContato contato={meuContato} />
    </div>
  );
}

export default App;



Dicas:
Quando você tem um objeto complexo como prop, a desestruturação pode ser usada para extrair as propriedades do objeto contato dentro de DadosContato para tornar o código mais limpo.
Sugestão de Extensão:
Crie um componente Endereco que receba um objeto endereco como prop (com rua, numero, cidade) e o exiba. Em seguida, inclua o Endereco dentro de DadosContato.
