Exercício 7: Componentização e Reuso

  • Descrição: Crie um componente Botao reutilizável.
  • Conteúdo Teórico: Podemos reutilizar componentes ao passar props diferentes.

Exemplo de Código:

function Botao(props) {
  return <button>{props.texto}</button>;
}

Dica: Renderize <Botao texto="Clique aqui" /> várias vezes.
Sugestão de Extensão: Adicione um estilo diferente para cada botão.
