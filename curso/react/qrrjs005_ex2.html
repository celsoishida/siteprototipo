Exercício 2: Componente de Classe Simples

  • Descrição: Crie um componente de classe chamado Relogio que renderize um elemento <h2> com a mensagem "Componente de Classe: Hora Atual". Não se preocupe com a hora real ainda.
  • Conteúdo Teórico:
    • Componentes de classe são classes JavaScript que estendem React.Component. Eles possuem um método render() obrigatório que retorna o JSX.
    • Historicamente, componentes de classe eram usados para gerenciar estado e ciclo de vida, mas com os Hooks, componentes funcionais podem fazer tudo isso e são geralmente preferidos. No entanto, é fundamental entender os componentes de classe, pois você os encontrará em muitos projetos existentes.

Dica:
Um componente de classe precisa importar React e estender React.Component. O método render() é onde você retorna o JSX.
Sugestão de Extensão:
Dentro do método render(), adicione um <span> exibindo a string "12:30:00" ao lado da mensagem.
