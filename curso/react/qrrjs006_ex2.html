Exercício 2: Exibindo Informações de Usuário

  • Descrição: Crie um componente chamado PerfilUsuario que receba duas props: nome e idade. Este componente deve exibir o nome e a idade do usuário em parágrafos separados.
  • Conceitos Teóricos:
    • Você pode passar múltiplas props para um componente. Cada prop é uma propriedade do objeto props.
    • É comum usar parênteses () ao retornar múltiplos elementos JSX para facilitar a leitura.

Exemplo de Código:

// App.js
import React from 'react';

function PerfilUsuario(props) {
  return (
    <div>
      <p>Nome: {props.nome}</p>
      <p>Idade: {props.idade}</p>
    </div>
  );
}

function App() {
  return (
    <div>
      <PerfilUsuario nome="Alice" idade={30} />
    </div>
  );
}

export default App;



Dicas:
Ao passar valores numéricos para props, use chaves {}. Por exemplo: idade={25}.
Sugestão de Extensão:
Adicione uma terceira prop chamada cidade e exiba-a no componente.
