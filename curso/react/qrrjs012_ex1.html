Exercício 1: Estilizando com CSS Tradicional

  • Descrição: Crie um componente React chamado BotaoSimples que exiba um botão. Estilize esse botão para ter um fundo azul, texto branco e bordas arredondadas usando um arquivo CSS tradicional (App.css).
  • Conceitos Teóricos:
    • CSS Tradicional: É a forma mais básica de estilizar aplicações web. Você cria um arquivo .css e importa-o globalmente na sua aplicação.
    • Classes CSS: Em React, você usa a propriedade className em vez de class para aplicar classes CSS aos elementos.

Exemplo de Código:
Crie um arquivo App.css com as seguintes regras:

.botao-simples {
background-color: #007bff;
color: white;
padding: 10px 20px;
border-radius: 5px;
border: none;
cursor: pointer;
}

No seu componente BotaoSimples.jsx (ou App.jsx):

import React from 'react';
import './App.css'; // Importe o arquivo CSS

function BotaoSimples() {
return (

);
}

export default BotaoSimples;



Dica:
Lembre-se de importar o arquivo CSS no componente onde você quer aplicar os estilos.
Sugestão de Extensão:
Adicione um estado de hover ao botão, mudando a cor de fundo quando o mouse passar por cima.
