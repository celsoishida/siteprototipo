Exercício 3: Input Controlado

  • Descrição: Crie um componente que contenha um campo de input de texto e um parágrafo. O texto digitado no input deve ser exibido em tempo real no parágrafo.
  • Conceitos Teóricos:
    • Input Controlado: No React, um "input controlado" é um elemento de formulário cujo valor é controlado pelo estado do React.
    • Evento onChange: Este evento é disparado sempre que o valor de um campo de input muda.

Dicas:
O evento onChange do input fornece o valor atualizado do campo através de event.target.value.
