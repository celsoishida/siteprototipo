Exercício 8: State com Objeto

  • Descrição: Crie um componente DadosDoProduto que exiba o nome, preço e estoque de um produto. Use um único state para armazenar todas essas informações em um objeto. Adicione inputs para cada propriedade que permitam a atualização dos dados do produto.
  • Conteúdo Teórico:
    • State com Objetos: É comum agrupar dados relacionados em um único objeto dentro do state. Ao atualizar um objeto no state, você deve criar um novo objeto com as propriedades atualizadas e usar a função de atualização do useState. Lembre-se de usar o operador spread (...) para copiar as propriedades existentes do objeto.

Exemplo de Código (Trecho de atualização de state):

const handleInputChange = (event) => {
  const { name, value } = event.target;
  setProduto(prevProduto => ({
    ...prevProduto,
    [name]: value
  }));
};



Dica:
Use o atributo name nos seus inputs para identificar qual propriedade do objeto deve ser atualizada.
Sugestão de Extensão:
Adicione um botão "Resetar" que redefina todos os dados do produto para seus valores iniciais.
