Exercício 6: Navegação Programática com useNavigate

  • Descrição: Na sua página `ProductDetail.js`, adicione um botão "Voltar para Produtos" que, ao ser clicado, navegue programaticamente de volta para a página `Products.js` usando o hook `useNavigate`.
  • Conceitos Teóricos:
    • useNavigate: Um hook que retorna uma função que você pode usar para navegar programaticamente pela aplicação. Útil para redirecionamentos após ações do usuário (ex: envio de formulário).

Exemplo de Código:

// src/components/ProductDetail.js (atualizado)
import React from 'react';
import { useParams, useNavigate } from 'react-router-dom'; // Importe useNavigate

function ProductDetail() {
  const { id } = useParams();
  const navigate = useNavigate(); // Obtenha a função de navegação

  const product = { id: id, name: `Produto ${id}`, description: `Detalhes do Produto ${id}.` };

  const handleGoBack = () => {
    navigate('/products'); // Navega para a rota /products
  };

  return (
    <div>
      <h2>Detalhes do Produto: {product.name}</h2>
      <p>ID: {product.id}</p>
      <p>{product.description}</p>
      <button onClick={handleGoBack}>Voltar para Produtos</button>
    </div>
  );
}

export default ProductDetail;



Dicas:
- `navigate()` pode receber um caminho como string ou um número para navegar para trás ou para frente no histórico do navegador (ex: `navigate(-1)` para voltar).

Sugestão de Extensão:
- Adicione um botão "Voltar" genérico em um componente de layout que use `navigate(-1)`.
