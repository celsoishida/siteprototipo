Exercício 6: Introdução ao Redux - Configurando a Store

  • Descrição: Configure uma store Redux básica. Crie um reducer simples para gerenciar um contador. A store deve ser capaz de incrementar e decrementar esse contador.
  • Conceitos Teóricos:
    • Redux: É uma biblioteca JavaScript para gerenciar o estado da aplicação. É frequentemente usada com React, mas não é exclusiva dele. Ajuda a criar aplicações que se comportam de forma consistente, executam em diferentes ambientes e são fáceis de testar.
    • Store: É o objeto que mantém o estado da aplicação. Existe apenas uma store por aplicação Redux.
    • Reducer: Uma função pura que recebe o estado atual e uma ação, e retorna o novo estado. É o único lugar onde a lógica de atualização do estado deve ocorrer.
    • Ação (Action): Um objeto JavaScript puro que descreve o que aconteceu. Possui uma propriedade type que indica o tipo de ação, e pode ter outras propriedades com o payload (dados) necessários.
    • createStore (Redux Legado) / configureStore (Redux Toolkit): Funções para criar a store Redux. Para iniciantes, é recomendado começar com o Redux Toolkit devido à sua simplicidade e boas práticas embutidas.

Exemplo de Código:

// store/index.js (usando Redux Toolkit)
import { configureStore, createSlice } from '@reduxjs/toolkit';

const counterSlice = createSlice({
  name: 'counter',
  initialState: { value: 0 },
  reducers: {
    increment: (state) => {
      state.value += 1;
    },
    decrement: (state) => {
      state.value -= 1;
    },
    incrementByAmount: (state, action) => {
      state.value += action.payload;
    },
  },
});

export const { increment, decrement, incrementByAmount } = counterSlice.actions;

export const store = configureStore({
  reducer: {
    counter: counterSlice.reducer,
  },
});

// index.js (ou main.jsx se usando Vite)
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App';
import { store } from './store';
import { Provider } from 'react-redux';

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <Provider store={store}>
      <App />
    </Provider>
  </React.StrictMode>,
);

// App.js
import React from 'react';
import { useSelector, useDispatch } from 'react-redux';
import { increment, decrement } from './store';

function App() {
  const count = useSelector((state) => state.counter.value);
  const dispatch = useDispatch();

  return (
    <div>
      <h1>Contador Redux: {count}</h1>
      <button onClick={() => dispatch(increment())}>Incrementar</button>
      <button onClick={() => dispatch(decrement())}>Decrementar</button>
    </div>
  );
}

export default App;



Dica:
O Redux Toolkit simplifica muito a configuração do Redux. createSlice define o nome, estado inicial e os reducers (que também geram as ações automaticamente).

Sugestão de Extensão:
Adicione um input para que o usuário possa incrementar o contador por uma quantidade específica (usando a ação incrementByAmount).
