Exercício 4: Listas com Props

  • Descrição: Crie um componente ListaDeItens que receba uma prop itens, que será um array de strings. O componente deve renderizar uma lista não ordenada (<ul>) onde cada item do array é um item de lista (<li>).
  • Conceitos Teóricos:
    • Você pode passar arrays como props.
    • Ao renderizar listas no React, é fundamental usar a prop key em cada item da lista. A key ajuda o React a identificar quais itens foram alterados, adicionados ou removidos, otimizando a performance. A key deve ser uma string única entre os irmãos.
    • O método map() de arrays é perfeito para transformar um array de dados em um array de elementos JSX.

Exemplo de Código:

// App.js
import React from 'react';

function ListaDeItens(props) {
  return (
    <ul>
      {props.itens.map((item, index) => (
        <li key={index}>{item}</li>
      ))}
    </ul>
  );
}

function App() {
  const frutas = ["Maçã", "Banana", "Laranja"];
  return (
    <div>
      <h3>Minhas Frutas:</h3>
      <ListaDeItens itens={frutas} />
    </div>
  );
}

export default App;



Dicas:
Quando não houver um ID único disponível, você pode usar o index como key, mas cuidado: isso não é recomendado se a lista puder ser reordenada ou ter itens adicionados/removidos no meio.
