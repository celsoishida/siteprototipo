VREA008 - Webscraping sobre venda de imóveis

Procurar informações sobre os imóveis à venda e criar uma base de dados para futura análise.

Plano de trabalho

  • Título do Plano de Trabalho: Levantamento da lista de portais; criar rotina de atualizar a lista dinamicamente; criar um rotina de webscraping para construir uma base de dados de imóveis para venda.
  • Objetivos: Criação de uma base para avaliação de preços
  • Metodologia/Síntese de Atividades: Criar um rotina de webscraping em busca de informações sobre os imóveis à venda, formatar uma base de dados para avaliação dos preços.
  • Cronograma: prazo até início Agosto/2025
  • Carga horária: 40 horas ou estágio supervisionado
  • Status: Aguardando voluntários.
