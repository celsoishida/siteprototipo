Curso: Introdução à programação web

  • Período: 07/04/2025 à 21/04/2025
  • CH: 18 horas
  • Objetivo geral
    • Capacitar os cursistas a desenvolverem páginas web interativas e responsivas, utilizando HTML5 e CSS3
  • Objetivos Específicos:
    • Estruturar Páginas Web: Utilizar HTML5 para organizar o conteúdo de forma sem ntica e acess vel.
    • Estilizar Layouts: Aplicar técnicas de CSS3 para criar páginas visualmente atraentes e responsivas.
    • Responsividade: Configurar layouts adaptáveis a diferentes dispositivos com CSS3.
    • Formulários Básicos: Implementar formulários funcionais com HTML5 e validar dados diretamente no navegador.
    • Animações Simples: Utilizar CSS3 para criar efeitos visuais básicos e interatividade.
