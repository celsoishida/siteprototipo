Cadastrar no SIGEU e Gerar PDF

  1. Acesse o SIGEU com seu email e senha. São os mesmos dados para acesso na intranet.
  2. Para abrir tela de cadastro acesse o relatório anual do Programa
  3. Caminho para chegar para a inserção do relatório é mostrado no primeiro vídeo a seguir
  4. É importante que o coordenador preencha todos os campos adequadamente. Dica: Oriente-se pela ficha de avaliação
    1. Após salvar, abrirão outros campos como Recursos humanos (quem trabalhou), participantes (alunos), financeiro, anexos (anexar a ata de aprovação no departamento).
  5. A equipe e suas horas devem estar devidamente cadastradas em recursos humanos. Veja como no vídeo: https://youtu.be/2JUL5nZsIg0
  6. Note que, se deixar o mouse sobre o campo, aparecerá uma explicação sobre o que deve ser inserido no mesmo
Observações:
  1. A partir do momento que o relatório estiver completamente preenchida, o coordenador deve clicar no ícone para gerar o relatório PDF.
  2. O coordenador envia o documento PDF para o chefe do departamento para aprovação em plenária departamental.
  3. Note que a submissão não é feita neste momento e nem é necessário a impressão para a assinatura da proposta. É importante não submeter para que as alterações sugeridas no departamento possam ser realizadas
