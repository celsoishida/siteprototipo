Relatório final

Cadastrar no SIGEU e gerar PDF

  1. Acesse o sigeu com seu email e senha. São os mesmos dados para acesso na intranet.
  2. Para abrir tela de cadastro acesse o relatório final do curso
  3. Caminho para chegar para a inserção do relatório é mostrado no primeiro vídeo a seguir (note que a criação do relatório é o mesmo para evento e curso)
  4. É importante que o coordenador preencha todos os campos adequadamente.
    1. Pergunta: "Ocorreu conforme o planejado?" Se sim, vá direto para o resumo. Caso contrário abrirá um campo 'Motivo' para indicar as explicações dos problemas e modificações da proposta.
    2. Pergunta: "Resumo". Não se esqueça dos resultados obtidos: número de participantes, total com direito a certificado
    3. Após salvar, abrirão outros campos como Recursos humanos (quem trabalhou), participantes (alunos), financeiro, anexos (anexar a ata de aprovação no departamento).
  5. A equipe e suas horas devem estar devidamente cadastradas em recursos humanos. Veja como no vídeo: https://youtu.be/2JUL5nZsIg0
  6. Cada participante devem ter o seu nome digitado em Participantes com o nome completo e, pelo menos, CPF para identificação do mesmo
  7. Note que, se deixar o mouse sobre o campo, aparecerá uma explicação sobre o que deve ser inserido no mesmo
  8. A carga horária da coordenação (coordenador mais vice) não deve ultrapassar o estabelecido). Mais informações veja a tabela no link
Observações:
  1. A partir do momento que a proposta estiver completamente preenchida, o coordenador deve clicar no ícone para gerar o documento PDF com a proposta.
  2. O coordenador deve enviar o documento PDF gerado para o chefe do departamento para aprovação em plenária departamental.
  3. Note que a submissão não é feita neste momento e nem é necessário a impressão para a assinatura da proposta. É importante não submeter para que as alterações do departamento possam ser realizadas
