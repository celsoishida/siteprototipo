Bem-vindo estudantes!

O estágio supervisionado é uma oportunidade de aplicar os seus conhecimentos para resolver problemas práticos dentro de um ambiente organizacional. O estágio é feito em duas etapas: estágio I para levantar os problemas e o diagnóstico de uma solução; estágio II é a implementação e relato dos resultados obtidos. O TCC é a formalização por escrito dos conhecimentos obtidos abordando um assunto específico. Os alunos de graduação que possuem resultados relavantes e inovadores são convidados a publicar como uma forma de melhoria do seu currículo.

Veja também instruções para escrita de documentos científicos.
